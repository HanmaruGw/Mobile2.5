DEXT5 Upload :: ShowFolderView

XML : <show_folder_view>.....</show_folder_view>, Config : config.ShowFolderView

Config 설정이 우선 적용됩니다.
업로드 폴더 올리기시 폴더로 진행할지, 파일로 보일지 설정할 수 있습니다.

remarks

기본값은 "1" 이고 폴더로 진행, "0"으로 설정시 폴더 올리기 시 파일로 보여집니다.

sample code

업로드 폴더 올리기시 파일로 보여지도록 설정합니다.

XML

.....
<setting>
    .....
    <show_folder_view>0</show_folder_view>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        DEXT5UPLOAD.config.ShowFolderView  = '0';

        new Dext5Upload("upload1");
    </script>
</div>
