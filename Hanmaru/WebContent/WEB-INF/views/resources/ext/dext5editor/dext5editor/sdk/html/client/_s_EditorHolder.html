DEXT5 Editor :: EditorHolder

config.EditorHolder

에디터를 특정 영역에서 생성할 수 있도록 합니다.

remarks

에디터를 특정 영역에서 생성하고자 하는 경우 생성할 영역의 id값을 입력합니다.
DEXT5.config.EditorHolder = "아이디값" 으로 설정합니다.

sample code

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>
	
<script type="text/javascript">

    // 에디터에 생성할 객체의 ID를 editor_area 로 설정합니다.
    function fn_load() { 
        DEXT5.config.EditorHolder = "editor_area";
        new Dext5editor("editor1");
    }

</script>

<!-- ..... 생략 ..... -->
 
<body onload="fn_load()">     
    
    <!-- ..... 생략 ..... -->       
    
    <div id="editor_area">

    <!-- 에디터를 생성할 영역입니다. -->

    </div> 
</body>
