DEXT5 Editor :: DialogWindow

config.DialogWindow

에디터 팝업을 Iframe 외부 최 상단 페이지로 설정합니다.

remarks

팝업창, 우클릭 메뉴를 띄우고 싶은 윈도우를 설정합니다 .

sample code

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        // 에디터 팝업을 parent.window 으로 설정합니다.
        DEXT5.config.DialogWindow = parent.window;

        new Dext5editor("editor1");

    </script>       
</div>
