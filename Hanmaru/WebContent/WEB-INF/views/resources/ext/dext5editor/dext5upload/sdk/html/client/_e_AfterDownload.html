DEXT5 Upload :: AfterDownload

void DEXT5UPLOAD_AfterDownload(uploadID, strDownloadPath, nViewOrSave)

다운로드 후 발생합니다.

return value

없음.

parameters

uploadID  다운로드가 발생한 업로드의 id를 의미합니다.
strDownloadPath  다운로드 된 파일의 전체 경로를 의미합니다.
nViewOrSave  열기일 경우 "0"이고, 다운로드일 경우 "1"를 의미합니다.(웹표준모드에서는 다운로드만 이벤트 발생)

remarks

DEXT5 Upload 설정 config.xml에서 use_after_download_event 값이 "1" 일 경우 발생합니다.
다운로드 파일이 여러개일 경우 각 파일마다 이벤트가 발생합니다.

        // 다운로드 후 처리 할 내용

        }
</script>
