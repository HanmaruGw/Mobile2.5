DEXT5 Upload :: SetSkinColor

void SetSkinColor(strColor1, strColor2, strColor3, strColor4, uploadID)

DEXT5UPLOAD의 색상을 사용자가 원하는 색상으로 설정합니다.

return value

없음.

parameters

uploadID  실행할 업로드의 id를 의미합니다.

remarks

strColor1  업로드창(상단/ 하단/ 하단 버튼 바탕색)
strColor2  업로드창(상단/ 하단 버튼 텍스트 색)
strColor3  전송창(상단/ 전체 전송률/ 전송 용량률/ 취소 버튼 바탕색)
strColor4  전송창(상단/ 취소 버튼 텍스트 색)
#을 포함한 hex값으로 입력합니다.

sample code

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>	
<script type="text/javascript">
       function setSkinColor() {
            DEXT5UPLOAD.SetSkinColor("#ff0000", "#f7f140", "#33e8f5", "#c2ffd0", "upload1");
        }
</script>	

<!-- ..... 생략 ..... -->
 
<button type="button" onclick="setSkinColor()">스킨 컬러</button>

<div style="width:800px;height:200px">  
        <script type="text/javascript">
            new Dext5Upload("upload1");
        </script>
</div>
