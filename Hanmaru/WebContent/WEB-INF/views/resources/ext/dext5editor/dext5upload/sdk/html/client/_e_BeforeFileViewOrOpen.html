DEXT5 Upload :: BeforeFileViewOrOpen

boolean DEXT5UPLOAD_BeforeFileViewOrOpen(uploadID, strWebFile, strItemKey, strItemOrgName, strItemUrlOrPath, strLargeFile)

파일 열기 전 발생합니다.

return value

boolean  true일 경우 파일을 열기하고, false일 경우 열기하지 않습니다.

parameters

uploadID  파일 열기 요청한 업로드의 id를 의미합니다.
strWebFile  Web파일과 Local 파일의 구분 값을 의미합니다. (0:local, 1:web)
strItemKey  파일의 Unique Key를 의미합니다.
strItemOrgName  파일의 원본 파일명을 의미합니다.
strItemUrlOrPath  파일의 저장위치를 의미합니다.
strLargeFile  파일 대용량를 의미합니다.

remarks

DEXT5 Upload 설정 config.xml에서 use_view_or_open_event 값이 1일 경우 발생합니다.

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>	

<script type="text/javascript">
        function DEXT5UPLOAD_BeforeFileViewOrOpen(uploadID, strWebFile, strItemKey, strItemOrgName, strItemUrlOrPath, strLargeFile) {
            // 파일 열기전 처리할 내용

            return true or false;
        }
</script>
