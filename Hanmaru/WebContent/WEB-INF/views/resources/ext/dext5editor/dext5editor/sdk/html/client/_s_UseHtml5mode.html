DEXT5 Editor :: UseHtml5mode

XML : <uploader_setting use_html5mode="....."></uploader_setting>

HTML5가 지원되는 브라우저에서 파일 업로드시 HTML5에서 지원되는 파일첨부기능 사용여부를 설정합니다.
HTML5에서 지원되는 파일첨부기능 : Drag&Drop으로 파일추가 가능

remarks

기본값은 "1"로 HTML5에서 지원되는 파일첨부기능을 사용,
"0"으로 설정 시 HTML5에서 지원되는 파일첨부기능을 사용 안함으로 설정합니다.

sample code

파일 업로드시 HTML5에서 지원되는 파일첨부기능 사용여부를 사용함으로 설정합니다.

.....
<uploader_setting use_html5mode="1">
    .....
</uploader_setting>
