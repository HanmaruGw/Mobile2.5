DEXT5 Editor :: showToolbar

void showToolbar(value, editor_id)

에디터의 툴바는 상단, 하단 두 개로 구성되어 있습니다.
이 설정은 에디터의 툴바를 표시하거나 숨기는 기능을 설정합니다.

return value

없음.

parameters

value  에디터 영역에 설정할 값을 의미합니다.
editor_id  접근할 에디터의 id를 의미합니다.

remarks

사용 용도에 따라서 툴바를 표시하지 않기를 원할 때 사용하면 유용 할 수 있습니다.
"0" 으로 설정 시 툴바를 모두 숨기고,
"1" 로 설정 시 상단 툴바를 표시합니다.
"2" 로 설정 시 하단 툴바를 표시합니다.
"3" 으로 설정 시 상단, 하단 툴바를 모두 표시합니다.

sample code

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>

    function showToolbar(){
        // 에디터 툴바를 모두 숨김으로 설정합니다.
        DEXT5.showToolbar(0, 'editor1');
    }

</script>
