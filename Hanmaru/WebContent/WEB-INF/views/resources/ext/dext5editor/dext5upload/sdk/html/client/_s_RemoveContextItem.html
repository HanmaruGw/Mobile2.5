DEXT5 Upload :: RemoveContextItem

XML : <remove_context_item>.....</remove_context_item>, Config : config.RemoveContextItem

Config 설정이 우선 적용됩니다.
업로드 context menu(우 클릭 메뉴)에서 삭제 할 항목을 설정합니다.

remarks

(공통)
파일추가 : add_file
선택된파일 제거 : remove_current_file
전체 파일 제거 : remove_all_file
열기 : open_current_file
선택된 파일 다운로드 : download_current_file
전체 다운로드 : download_all_file
맨 앞으로 : move_first
앞으로 : move_forward
뒤로 : move_backward
맨 뒤로 : move_end

(플러그인 전용)
폴더로 파일 추가 : add_files_from_folder
폴더추가 : add_folder
이미지 붙여넣기 : image_paste
다운로드 후 열기 : download_and_open
인쇄 : print
설정 : setting

항목 입력 구분자는 ",(콤마) " 입니다.

sample code

context menu(우 클릭 메뉴)에서 add_file(파일추가) 항목을 삭제합니다.

XML

.....
<setting>
    .....
    <remove_context_item>add_file</remove_context_item>
    .....
</setting>
.....
