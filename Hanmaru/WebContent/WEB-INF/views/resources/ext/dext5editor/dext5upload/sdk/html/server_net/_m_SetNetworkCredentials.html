DEXT5 Upload :: SetNetworkCredentials

void SetNetworkCredentials(string userID, string userPassword, string domain)

파일을 업로드 하려는 위치가 계정이 있는 네트워크 드라이브이거나,
다운로드 하려는 파일의 위치가 계정이 있는 네트워크 드라이브일 경우
네트워크 드라이브의 인증 정보를 설정해야 합니다.

return value

없음.

parameters

userID  Network Drive 인증 ID를 의미합니다.
userPassword  Network Drive 인증 Password를 의미합니다.
domain  Network Drive 접속 Domain을 의미합니다.

remarks

없음.

sample code

Raonwiz.Dext5.UploadHandler upload = new Raonwiz.Dext5.UploadHandler();

upload.SetNetworkCredentials("ID", "Password", @"\\Domain");

upload.Process(context);
