DEXT5 Editor :: setEditorMode

void setEditorMode(mode, editor_id)

에디터를 view 또는 edit 모드로 변경합니다.

return value

없음.

parameters

mode  설정할 모드를 의미합니다.
editor_id  접근할 에디터의 id를 의미합니다.

remarks

읽기전용으로 사용하면서 수정이 필요할 때 유용합니다.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
<script type="text/javascript">

    function setViewMode() {
        // 에디터를 view 모드로 설정합니다.
        DEXT5.setEditorMode('view', 'editor1');
    }

    function setEditMode() {
        // 에디터를 edit 모드로 설정합니다.
        DEXT5.setEditorMode('edit', 'editor1');
    }

</script>	

<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">
  
    <!-- 에디터 생성 -->  
    <script type="text/javascript">
        new Dext5editor('editor1');
    </script>

</div>
