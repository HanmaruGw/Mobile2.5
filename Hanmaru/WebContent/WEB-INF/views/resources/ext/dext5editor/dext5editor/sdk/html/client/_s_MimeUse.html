DEXT5 Editor :: MimeUse

XML : <mime_use>.....</mime_use>, Config : config.MimeUse

Config 설정이 우선 적용됩니다.
MIME 사용 여부를 설정합니다.

remarks

기본값은 "0" 이고 MIME 미사용, "1" 로 설정 시 사용입니다.

sample code

MIME 을 사용으로 설정합니다.

XML

.....
<license>
    .....
    <mime_use>1</mime_use>
    .....
</license>
.....

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>
