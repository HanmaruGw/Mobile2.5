DEXT5 Upload :: EncryptParam

XML : <security><encrypt_param>.....</encrypt_param></security>, Config : config.Security.EncryptParam

Config 설정이 우선 적용됩니다.
전송하는 파라미터 암호화 여부를 설정합니다.

remarks

기본값은 "0" 미사용이고, "1"로 설정 시 전송하는 파라미터를 암호화합니다.

sample code

전송 파라미터 암호화를 사용으로 설정합니다.

XML

.....
<setting>
    .....
    <security>
        <encrypt_param>1</encrypt_param>
    </security>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        DEXT5UPLOAD.config.Security = {
            EncryptParam: "1"
        }

        new Dext5Upload("upload1");
    </script>
</div>
