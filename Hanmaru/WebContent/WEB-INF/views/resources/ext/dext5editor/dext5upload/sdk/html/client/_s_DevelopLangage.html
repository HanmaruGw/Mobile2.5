DEXT5 Upload :: DevelopLangage

XML : <develop_langage>.....</develop_langage>, Config : config.DevelopLangage

Config 설정이 우선 적용됩니다.
사용중인 사이트의 개발 언어를 설정합니다.

remarks

기본값은 "NET" 이고, NET, JAVA 중 하나의 값을 사용합니다.

sample code

사용중인 사이트 언어를 JAVA로 설정합니다.

XML

.....
<uploader_setting>
    .....
    <develop_langage>JAVA</develop_langage>
    .....
</uploader_setting>
.....

Config

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        DEXT5UPLOAD.config.DevelopLangage  = 'JAVA';

        new Dext5Upload("upload1");
    </script>
</div>
