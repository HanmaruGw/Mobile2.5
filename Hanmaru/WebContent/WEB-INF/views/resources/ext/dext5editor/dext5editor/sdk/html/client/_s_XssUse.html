DEXT5 Editor :: XssUse

XML : <xss_use>.....</xss_use>, Config : config.XssUse

Config 설정이 우선 적용됩니다.
에디터 XSS 방지를 설정합니다.
script, vbscript 태그 입력 시 문자열로 처리하고, src, href 등에 입력된 보안위협요소를 제거합니다.

remarks

기본값은 "1" 이고 XSS 방지 설정, "0" 으로 설정 시 XSS 방지 미설정 합니다.

sample code

에디터 XSS 방지 설정 합니다.

XML

.....
<xss_protection>
    <xss_use>1</xss_use>
    .....
</xss_protection>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.XssUse = "1";

        new Dext5editor("editor1");

    </script>       
</div>
