DEXT5 Editor :: XssRemoveEvents

XML : <xss_remove_events>.....</xss_remove_events>, Config : config.XssRemoveEvents

Config 설정이 우선 적용됩니다.
에디터에 입력되면 삭제할 이벤트를 설정합니다.

remarks

onclick, onload, onchange 등의 형태로 이벤트를 설정합니다.
all 로 입력하면 모든 이벤트 값을 설정합니다.
사용자가 태그에 해당되는 이벤트를 입력한 경우 에디터에서 삭제합니다.

sample code

에디터에 onclick이 입력되면 삭제합니다.

XML

.....
<xss_protection>
    .....
    <xss_remove_events>onclick</xss_remove_events>
    .....
</xss_protection>
.....
