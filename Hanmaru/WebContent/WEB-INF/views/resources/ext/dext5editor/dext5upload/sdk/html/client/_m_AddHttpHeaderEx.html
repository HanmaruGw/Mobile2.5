DEXT5 Upload :: AddHttpHeaderEx

void AddHttpHeaderEx(headerName, headerValue, uploadID)

HTTP Header의 값을 설정합니다.

return value

없음.

parameters

headerName  설정할 header 이름을 의미합니다.
headerValue  설정할 값을 의미합니다.
uploadID  실행할 업로드의 id를 의미합니다.

remarks

없음.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>	
<script type="text/javascript">
        // DEXT5UPLOAD 객체 생성 완료이벤트에서 설정하시면 됩니다.
        function DEXT5UPLOAD_OnCreationComplete(uploadID) {
            DEXT5UPLOAD.AddHttpHeaderEx("headerName", "headerValue", "upload1");
        }
</script>

1. javascript Cookie Setting

DEXT5UPLOAD.AddHttpHeaderEx("Cookie", document.cookie, "upload1");

2. jsp Cookie Setting

<%
    String strCookie = "";
    Cookie[] cookies = request.getCookies();
    int iCookieLength = cookies.length;
    for(int i = 0 ; i<iCookieLength; i++) {
        strcookie += cookies[i].getName() + "=" + cookies[i].getvalue() + ";";
    }
%>

DEXT5UPLOAD.AddHttpHeaderEx("Cookie", "<%=strCookie%>", "upload1");

3. aspx Cookie Setting

    string strCookie = "";
    int iCookieLength = Request.Cookies.Count;
    for (int i = 0; i < iCookieLength; i++)
    {
        strCookie += Request.Cookies[i].Name + "=" + Request.Cookies[i].Value + ";";

%>
