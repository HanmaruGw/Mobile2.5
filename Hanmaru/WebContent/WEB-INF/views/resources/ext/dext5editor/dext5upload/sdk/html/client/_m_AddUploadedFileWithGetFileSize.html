DEXT5 Upload :: AddUploadedFileWithGetFileSize

void AddUploadedFileWithGetFileSize
(FileUniqueKey, FileName, FilePath, FileSizePath, CustomValue, UploadId)

파일의 실제용량을 알수 없는 경우 사용하는 API로 파일 추가시 서버측에서 파일의 용량을 구하여 업로드에 추가합니다.

return value

없음.

parameters

FileUniqueKey  추가 할 파일의 Unique Key를 의미합니다.
FileName  추가 할 파일의 이름을 의미합니다.
FilePath  추가 할 파일의 저장위치를 의미합니다.(파일의 물리적 경로 또는 가상경로로 설정)
FileSizePath  추가 할 파일의 저장위치를 의미합니다. (FilePath의 경로가 실제 파일의 경로가 아닌경우 이 파라미터에 실제 경로를 입력합니다.)
CustomValue  Custom Download 사용시 사용될 값을 의미합니다.
UploadId  파일을 추가 할 업로드 객체의 id를 의미합니다.

remarks

없음.
