DEXT5 Upload :: InitXml

config.InitXml

업로드 생성시 로드할 XML파일을 설정합니다.

remarks

기본값은 "dext5upload.config.xml" 입니다.
파일 경로는 "dext5upload/config" 입니다.

sample code

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        // 업로드를 example.xml을 로드하여 생성합니다.
        DEXT5UPLOAD.config.InitXml = 'example.xml';

        new Dext5Upload("upload1");
    </script>
</div>
