DEXT5 Upload :: SetUploadedFile

void SetUploadedFile(ItemIndex, FileUniqueKey, FileName, FilePath, FileSize, CustomValue, UploadId)

업로드 리스트 컨트롤에 이미 업로드 되어 있는 웹파일의 정보를 변경합니다.

return value

없음.

parameters

ItemIndex  0부터 시작하는 정보를 변경할 파일의 순서를 의미합니다.
FileUniqueKey  변경 할 파일의 Unique Key를 의미합니다.
FileName  변경 할 파일의 이름을 의미합니다.
FilePath  변경 할 파일의 저장위치를 의미합니다.(파일의 물리적 경로 또는 가상경로로 설정)
FileSize  변경 할 파일의 크기를 의미합니다.
CustomValue  변경할 Custom Download 사용시 사용될 값을 의미합니다.
UploadId  파일정보를 변경 할 업로드 객체의 id를 의미합니다.

remarks

없음.
