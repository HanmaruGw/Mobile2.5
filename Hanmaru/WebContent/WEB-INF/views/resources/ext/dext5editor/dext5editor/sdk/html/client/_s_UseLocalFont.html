DEXT5 Editor :: UseLocalFont

XML : <font_family use_local_font="....."></font_family>, Config : config.UseLocalFont

Config 설정이 우선 적용됩니다.
에디터 글꼴 메뉴에 사용자 로컬 폰트 추가 여부를 설정합니다.
Internet Explorer 브라우저에서만 지원됩니다.(Edge 브라우저 제외)

remarks

기본값은 "0" 이고 사용자 로컬 폰트 추가 미설정, "1" 로 설정 시 사용자 로컬 폰트 추가를 설정 합니다.

sample code

에디터 글꼴 메뉴에 사용자 로컬 폰트 추가 설정 합니다.

XML

.....
<font_family use_local_font="1">
    .....
</font_family>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.UseLocalFont = "1";

        new Dext5editor("editor1");

    </script>       
</div>
