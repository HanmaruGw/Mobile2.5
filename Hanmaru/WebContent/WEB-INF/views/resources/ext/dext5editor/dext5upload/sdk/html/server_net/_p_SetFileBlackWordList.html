DEXT5 Upload :: SetFileBlackWordList

upload.SetFileBlackWordList

업로드시 서버에서 파일명에대한 제어를 위하여 설정합니다.
설정된 파일명의 파일이 업로드 될 경우 Error가 발생됩니다.

remarks

기본값 : 없음.

sample code

Raonwiz.Dext5.UploadHandler upload = new Raonwiz.Dext5.UploadHandler();

string[] tempWordList  = new string[1];
tempWordList[0] = "hacking";
upload.SetFileBlackWordList = tempWordList;

upload.Process(context);
