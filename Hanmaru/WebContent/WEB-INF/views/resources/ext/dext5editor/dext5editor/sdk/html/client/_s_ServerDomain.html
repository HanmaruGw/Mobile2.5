DEXT5 Editor :: ServerDomain

XML : <server_domain>.....</server_domain>, Config : config.ServerDomain

Config 설정이 우선 적용됩니다.
이미지, 미디어 파일을 올릴 때 웹 경로에 들어가는 업로드 된 파일의 도메인 부분을 설정합니다.

remarks

"빈 값"인 경우 handler_url이 동작하는 서버의 도메인이 설정됩니다.
"/" 값을 지정하면 도메인 부분을 빼고 처리합니다.
"/" 이외의 값을 지정하면 해당 값으로 도메인이 표시됩니다.

ex) handler_url 동작 URL이 http://www.dext5.com/upload_handler.ashx 인 경우
     (빈값) ==> http://www.dext5.com/dext5editordata/image.jpg
     "/" ==> dext5editordata/image.jpg
     "http://dext5.com/" ==> http://dext5.com/dext5editordata/image.jpg

sample code

웹 경로에 들어가는 업로드 된 파일의 도메인 부분을 "/"로 설정합니다.

XML

.....
<uploader_setting>
    .....
    <server_domain>/</server_domain>
    .....
</uploader_setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">  
    <script type="text/javascript">

        DEXT5.config.ServerDomain = "/";

        new Dext5editor("editor1");

    </script>       
</div>
