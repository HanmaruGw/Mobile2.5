DEXT5 Editor :: UseLocalFont

XML : <font_family use_recently_font="....."></font_family>, Config : config.UseRecentlyFont

Config 설정이 우선 적용됩니다.
에디터 글꼴 메뉴에 최근사용 폰트 추가 여부를 설정합니다.
제일 상단에 최대 5개까지 표현됩니다.

remarks

기본값은 "0" 이고 최근사용 폰트 추가 미설정, "1" 로 설정 시 최근사용 폰트 추가를 설정 합니다.

sample code

에디터 글꼴 메뉴에 최근사용 폰트 추가 설정 합니다.

XML

.....
<font_family use_recently_font="1">
    .....
</font_family>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.UseRecentlyFont = "1";

        new Dext5editor("editor1");

    </script>       
</div>
