DEXT5 Editor :: ImageConvertWidth

XML : <image_convert_width>.....</image_convert_width>, Config : config.ImageConvertWidth

Config 설정이 우선 적용됩니다.
이미지파일 업로드 시 저장될 이미지의 너비를 변환 할 때 사용하는 옵션입니다.

remarks

기본값은 "0" 입니다.
image_convert_width가 빈 값 또는 "0" 이고,
image_convert_height 의 값이 0 보다 큰 정수일 때 입력 값을 기준으로 원본 비율의 width 값을 계산하여 jpg로 변환합니다.

sample code

이미지파일 업로드 시 저장될 이미지의 너비를 100으로 변환합니다.

XML

.....
<uploader_setting>
    .....
    <image_convert_width>100</image_convert_width>
    .....
</uploader_setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">   
    <script type="text/javascript">

        DEXT5.config.ImageConvertWidth = "100";

        new Dext5editor("editor1");

    </script>       
</div>
