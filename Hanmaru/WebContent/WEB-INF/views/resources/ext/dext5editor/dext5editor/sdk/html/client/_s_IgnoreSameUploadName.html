DEXT5 Upload :: IgnoreSameUploadName

config.IgnoreSameUploadName

업로드 명 중복시 임의로 변경 생성하도록 설정합니다.

remarks

기본값은 "0" 이고, "1" 로 설정 시 중복된 업로드 명이 발생 할 경우 업로드에서 임의로 변경 합니다. (단, 동적 생성 시)

sample code

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        // 중복 된 업로드 명이 발생 할 경우 임의로 변경 설정합니다.
        DEXT5UPLOAD.config.IgnoreSameUploadName = "1";

        new Dext5Upload("upload1");
    </script>
</div>
