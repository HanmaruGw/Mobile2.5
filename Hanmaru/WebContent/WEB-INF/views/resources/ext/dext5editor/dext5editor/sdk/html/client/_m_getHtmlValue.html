DEXT5 Editor :: getHtmlValue

string getHtmlValue(editor_id)

에디터 디자인 영역의 HTML 소스에서 <html> 태그 내부의 소스를 가져옵니다.
일반적으로 디자인 영역을 직접 접근할 때 유용합니다.
(이 함수를 사용하시려면 Editor에 글을 적용할 때 setHtmlValue 함수를 사용해야 합니다.)

return value

<html> 태그 내부의 소스를 리턴합니다.

parameters

editor_id  html 소스를 가져올 에디터의 id를 의미합니다.

remarks

에디터를 1개만 생성했을 경우 parameter를 입력하지 않아도 html소스를 가져올수 있습니다.
예) id가 editor1인 에디터를 생성했을 경우
     DEXT5.getHtmlValue('editor1'), DEXT5.getHtmlValue() 모두 editor1 에디터 디자인 영역의 html 소스를 가져옵니다.

에디터를 2개 이상 생성했을 경우 parameter를 반드시 입력해야 합니다.
예) id가 editor1, editor2인 에디터를 생성했을 경우
     DEXT5.getHtmlValue('editor1')  editor1 에디터 디자인 영역의 html 소스를 가져옵니다.
     DEXT5.getHtmlValue('editor2')  editor2 에디터 디자인 영역의 html 소스를 가져옵니다.
