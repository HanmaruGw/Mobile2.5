DEXT5 Editor :: AllowInoutdentText

XML : <table_property allow_inoutdent_text="....."></table_property>
Config : config.AllowInoutdentText

Config 설정이 우선 적용됩니다.
표 전체 들여쓰기, 내어쓰기 또는 표 안에 텍스트만 들여쓰기, 내어쓰기 사용 여부를 설정합니다.

remarks

기본값은 "1" 이고 표 안에 텍스트만 들여쓰기, 내어쓰기 사용,
"0"으로 설정 시 표 전체 들여쓰기, 내어쓰기 사용으로 설정합니다.

sample code

들여쓰기, 내어쓰기시 표 전체 들여쓰기, 내어쓰기 사용으로 설정합니다.

XML

.....
<setting>
    .....
    <table_property allow_inoutdent_text="0">
        .....
    </table_property>
    .....
</setting>
.....
