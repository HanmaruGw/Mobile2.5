DEXT5 Editor :: StatusBarInitMode

XML : <status_bar init_mode="....."></status_bar>, Config : config.StatusBarInitMode

Config 설정이 우선 적용됩니다.
에디터 로드 시 보여질 보기화면(statusBar) 모드를 설정합니다.

remarks

디자인은 "design", HTML은 "source", 미리보기는 "preview" ,TEXT는 "text" 로 설정합니다.
기본값은 "design" 입니다.
