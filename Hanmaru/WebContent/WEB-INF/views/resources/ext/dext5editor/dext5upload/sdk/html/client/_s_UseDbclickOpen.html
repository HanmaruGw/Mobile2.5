DEXT5 Upload :: UseDbclickOpen

XML : <use_dbclick_open>.....</use_dbclick_open>, Config : config.UseDbclickOpen

Config 설정이 우선 적용됩니다.
더블클릭 시 파일열기 사용을 설정합니다.

remarks

기본값은 "1" 사용이고, "0"으로 설정시 파일을 더블클릭 시 파일열기를 사용하지 않습니다.

sample code

더블클릭 시 파일열기를 미사용으로 설정합니다.

XML

.....
<setting>
    .....
    <use_dbclick_open>0</use_dbclick_open>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        DEXT5UPLOAD.config.UseDbclickOpen = '0';

        new Dext5Upload("upload1");
    </script>
</div>
