DEXT5 Editor :: XssAllowEventsAttribute

XML : <xss_use allow_events_attribute="....."></xss_use>, Config : config.XssAllowEventsAttribute

Config 설정이 우선 적용됩니다.
객체의 Attribute에 설정된 이벤트들을 허용할지 여부를 설정합니다.

remarks

기본값은 "1" 이고 객체의 Attribute에 설정된 이벤트들을 허용 설정,
"0" 으로 설정 시 객체의 Attribute에 설정된 이벤트들을 허용하지 않습니다.

sample code

객체의 Attribute에 설정된 이벤트들을 허용 설정 합니다.

XML

.....
<xss_protection>
    <xss_use allow_events_attribute="1">.....</xss_use>
    .....
</xss_protection>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>
