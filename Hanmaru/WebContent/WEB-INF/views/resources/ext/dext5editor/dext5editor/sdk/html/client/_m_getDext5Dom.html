DEXT5 Editor :: getDext5Dom

Dom getDext5Dom(editor_id)

디자인 영역의 documentElement Dom을 가져옵니다.
일반적으로 디자인 영역을 직접 접근할 때 유용합니다.

return value

디자인 영역의 documentElement Dom을 리턴합니다.

parameters

editor_id  Dom을 가져올 에디터의 id를 의미합니다.

remarks

에디터를 1개만 생성했을 경우 parameter를 입력하지 않아도 Dom을 가져올 수 있습니다.
예) id가 editor1인 에디터를 생성했을 경우
     DEXT5.getDext5Dom('editor1'), DEXT5.getDext5Dom() 모두 editor1 에디터 디자인 영역의 Dom을 가져옵니다.

에디터를 2개 이상 생성했을 경우 parameter를 반드시 입력해야 합니다.
예) id가 editor1, editor2인 에디터를 생성했을 경우
     DEXT5.getDext5Dom('editor1')  editor1 에디터 디자인 영역의 Dom을 가져옵니다.
     DEXT5.getDext5Dom('editor2')  editor2 에디터 디자인 영역의 Dom을 가져옵니다.
