[관리자 라이선스 전용]

DEXT5 Editor :: dext_editor_textarea_event

boolean dext_editor_textarea_event(element, editorId)

관리자모드에서 textarea태그로 추가된 객체를 더블클릭 시 이벤트가 발생합니다.

return value

true를 리턴할 경우 속성 다이얼로그창이 나타나고, false를 리턴할 경우 나타나지 않습니다.

parameters

element  이벤트가 발생한 요소입니다.
editorId  이벤트가 발생한 에디터 id입니다.

remarks

에디터의 설정파일(dext5editor/config/dext_editor.xml)의 manager_mode 설정의 use 속성값이 1인 경우에 이벤트가 발생합니다.
관리자모드에서 textarea태그를 추가하는 기능은 입력영역입니다.
