DEXT5 Upload :: InitServerXml

config.InitServerXml

만일 업로드 생성시 로드할 XML에 파일 저장 경로등과 같은 값의 노출이 문제가 되는 경우
XML파일을 암호화 하여 처리 하는 방식을 의미합니다.

remarks

기본값은 빈값으로 기본 XML을 읽는 것 입니다. ("dext5upload/config/dext5upload.config.xml")

sample code

Client

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        // Full URL(http 포함)로 다음과 같이 설정합니다.
        // Full URL은 DEXT5 Upload Handler경로로 지정하며, f 파라미터의 값은 서버에서 처리할 XML 파일명을 지정합니다.
        DEXT5UPLOAD.config.InitServerXml = 'http://localhost/dext5upload/handler/dext5handler.jsp?f=dext5upload.config.xml';

        new Dext5Upload("upload1");
    </script>
</div>

Server

.NET

upload.SetConfigPhysicalPath = "D:\\config"; // 환경설정파일(xml)이 존재하는 서버의 폴더 경로

JAVA

upload.SetConfigPhysicalPath("D:/config"); // 환경설정파일(xml)이 존재하는 서버의 폴더 경로
