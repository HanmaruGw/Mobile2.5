DEXT5 Upload :: SetFileInfo

void SetFileInfo(preFix, subFix, uploadID)

파일명에 붙일 특정 문자열을 설정합니다.

return value

없음.

parameters

preFix  파일명 앞에 붙일 문자열을 의미합니다.
subFix  파일명 뒤에 붙일 문자열을 의미합니다.
uploadID  특정 문자열을 붙일 업로드의 id를 의미합니다.

remarks

SetFileInfo 함수를 이용하여 파일명을 변경하기 위해서는 서버에서의 작업이 필요합니다.
DEXT5 Upload 개발자매뉴얼 > 기타가이드 > 파라미터 추가 후 서버에서 받는 방법, 파일을 원하는 파일명으로 변환하는 방법을 참고하시어
핸들러의 파일 저장 전 이벤트에서 처리하시기 바랍니다.
핸들러에서 다음과 같이 preFix, subFix값을 받습니다.

string[] preFix = upload.GetRequestValue(context, "d5_prefix");
string[] subFix = upload.GetRequestValue(context, "d5_subfix");

sample code

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>	
<script type="text/javascript">
        DEXT5UPLOAD_OnCreationComplete(uploadID)
            DEXT5UPLOAD.SetFileInfo("preValue", "subValue", "upload1");

            // 핸들러에서 다음과 같이 preFix, subFix값을 받습니다.
            // string[] preFix = upload.GetRequestValue(context, "d5_prefix"); // preFix[0] == "preValue"
            // string[] subFix = upload.GetRequestValue(context, "d5_subfix"); // subFix[0] == "subValue"
        }
</script>
