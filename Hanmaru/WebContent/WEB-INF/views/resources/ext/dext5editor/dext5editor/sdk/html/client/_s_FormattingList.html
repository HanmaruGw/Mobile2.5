DEXT5 Editor :: FormattingList

config.FormattingList

에디터 기본 서식 메뉴에 노출할 서식 리스트를 설정합니다.

remarks

없음.

sample code

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">  
    <script type="text/javascript">

        // 에디터 서식 메뉴에 노출할 서식을 h1, h2 으로 설정합니다.
        DEXT5.config.FormattingList = "h1,h2";

        new Dext5editor("editor1");

    </script>       
</div>
