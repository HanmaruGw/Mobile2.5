DEXT5 Upload :: ViewerHandlerUrl

XML : <viewer_handler_url>.....</viewer_handler_url>, Config : config.ViewerHandlerUrl

Config 설정이 우선 적용됩니다.
웹표준 모드에서 열기 또는 파일리스트를 더블클릭하여 열리는 Viewer페이지를 구현하여 사용하고자 할 때 url을 설정합니다.

remarks

설정하실 때 http:// 부터의 전체경로를 설정합니다.

sample code

업로드 Viewer handler url을 설정합니다.

XML

.....
<uploader_setting>
    .....
    <viewer_handler_url>http://www.dext5.com/handler/dext5viewer.jsp</viewer_handler_url>
    .....
</uploader_setting>
.....

Config

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        DEXT5UPLOAD.config.ViewerHandlerUrl = 'http://www.dext5.com/handler/dext5viewer.jsp';

        new Dext5Upload("upload1");
    </script>
</div>
