DEXT5 Editor :: ToSaveFilePathURL

XML : <to_save_file_path_url>.....</to_save_file_path_url>, Config : config.ToSaveFilePathURL

Config 설정이 우선 적용됩니다.
이미지 또는 파일 삽입 등 저장 할 때 서버에 연결 된 저장소의 URL 경로를 지정합니다.

remarks

기본값은 dext5editor 폴더의 아래에 "dext5editordata" 폴더가 됩니다.
에디터에서 등록된 파일들은 설정된 폴더의 아래에 저장이 됩니다.
이 경로는 서버의 상대 경로 또는 루트(/) 절대 경로만 허용합니다.
SaveFileFolderNameRule 과 같이 사용할 수 있습니다.

sample code

dext5editor의 폴더가 dext5_sample이라는 폴더 아래에 있다고 가정할 경우
업로드된 파일을 dext5_sample/dext5editordata/user 폴더 아래에 저장합니다.

XML

.....
<uploader_setting>
    .....
    <to_save_file_path_url>dext5editordata/user</to_save_file_path_url>
    .....
</uploader_setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px"> 
        <script type="text/javascript">

        DEXT5.config.ToSaveFilePathURL = "dext5editordata/user";

        new Dext5editor("editor1");

    </script>                               
</div>
