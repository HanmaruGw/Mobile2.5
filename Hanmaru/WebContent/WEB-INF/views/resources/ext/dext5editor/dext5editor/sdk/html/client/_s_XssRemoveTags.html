DEXT5 Editor :: XssRemoveTags

XML : <xss_remove_tags>.....</xss_remove_tags>, Config : config.XssRemoveTags

Config 설정이 우선 적용됩니다.
에디터에 입력되면 삭제할 태그를 설정합니다.

remarks

script, iframe, object 등의 형태로 태그를 설정합니다.
사용자가 해당되는 태그를 입력한 경우 에디터에서 삭제합니다.

sample code

에디터에 script가 입력되면 삭제합니다.

XML

.....
<xss_protection>
    .....
    <xss_remove_tags>script</xss_remove_tags>
    .....
</xss_protection>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>
