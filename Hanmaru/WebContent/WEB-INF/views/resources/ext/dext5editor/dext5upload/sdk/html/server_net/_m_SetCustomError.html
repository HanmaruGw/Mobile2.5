DEXT5 Upload :: SetCustomError

void SetCustomError(string errorMessage)
void SetCustomError(string errorCode, string errorMessage)

업로드 또는 다운로드시 서버에서 특정 상황에 따른 에러처리를 할 경우 사용됩니다.
UploadBeforeInitialize, UploadCompleteBefore, OpenDownloadBeforeInitialize 이벤트에서 사용이 가능합니다.

return value

없음.

parameters

errorCode  Error에 대한 Code를 의미합니다. (900이상의 3자리로 설정)
errorMessage  Error에 대한 Message를 의미합니다.

remarks

errorCode 기본값 : 999

sample code

upload.SetCustomError("사용자오류");
upload.SetCustomError("999", "사용자오류");
