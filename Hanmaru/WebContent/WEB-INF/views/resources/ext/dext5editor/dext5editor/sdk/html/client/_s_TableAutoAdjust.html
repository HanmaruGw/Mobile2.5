DEXT5 Editor :: TableAutoAdjust

XML : <table_auto_adjust>.....</table_auto_adjust>, Config : config.TableAutoAdjust

Config 설정이 우선 적용됩니다.
에디터에서 표 작성 및 수정 시 width, height를 자동 보정으로 설정합니다.

remarks

"1"로 설정 시 사용으로 설정하고, "0" 으로 설정 시 사용 안함으로 설정합니다.
기본값은 "1" 입니다.

sample code

에디터에서 표 작성 및 수정 시 width, height를 자동 미보정으로 설정합니다.

XML

.....
<setting>
    .....
    <table_auto_adjust>0</table_auto_adjust>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
     
<!-- ..... 생략 ..... -->
     
<div style="width:900px;height:550px">  
    <script type="text/javascript">

        DEXT5.config.TableAutoAdjust = "0";
    
        new Dext5editor("editor1");

    </script>       
</div>
