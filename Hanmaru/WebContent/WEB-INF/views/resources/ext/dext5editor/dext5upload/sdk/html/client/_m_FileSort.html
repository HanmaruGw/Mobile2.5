DEXT5 Upload :: FileSort

void FileSort(sortType, sortMethod, uploadID)

업로드의 파일리스트를 정렬합니다.

return value

없음.

parameters

sortType  0이면 파일명을 기준, 1이면 파일사이즈를 기준, 2면 사용자정의헤더를 사용 시 사용자정의헤더를 기준
sortMethod  0이면 오름차순, 1이면 내림차순
uploadID  파일리스트를 정렬하는 업로드의 id값을 의미합니다.

remarks

없음.
