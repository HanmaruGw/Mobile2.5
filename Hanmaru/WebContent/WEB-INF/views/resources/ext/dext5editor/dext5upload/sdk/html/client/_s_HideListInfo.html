DEXT5 Upload :: HideListInfo

XML : <hide_list_info>.....</hide_list_info>, Config : config.HideListInfo

Config 설정이 우선 적용됩니다.
업로드 편집모드일 때 업로드 영역 중앙에 보여지는 가이드 문구를 표시하거나 숨기는 기능을 설정합니다.

remarks

기본값은 "0" 이고, "1" 로 설정 시 가이드 문구를 미노출 합니다.

sample code

업로드 영역에 보여지는 가이드 텍스트를 미표시 설정합니다.

XML

.....
<setting>
    .....
    <hide_list_info>1</hide_list_info>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        DEXT5UPLOAD.config.HideListInfo = '1';

        new Dext5Upload("upload1");
    </script>
</div>
