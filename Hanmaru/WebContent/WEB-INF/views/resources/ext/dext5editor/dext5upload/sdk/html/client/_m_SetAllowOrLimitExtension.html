DEXT5 Upload :: SetAllowOrLimitExtension

void SetAllowOrLimitExtension(allowOrLimit, extension, uploadID)

업로드 할 파일 확장자를 허용 또는 제한으로 설정합니다.

return value

없음.

parameters

allowOrLimit  업로드 할 파일 확장자를 허용할지 제한할지를 의미합니다. 0: 제한으로 설정, 1: 허용으로 설정
extension  제한 또는 허용할 파일 확장자를 의미합니다.(확장자가 여러개일 경우 콤마(,)로 구분하여 설정합니다)
uploadID  실행할 업로드의 id를 의미합니다.

remarks

없음.

sample code

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>	
<script type="text/javascript">
        function setAllowOrLimitExtension() {
            // gif,jpg 확장자의 업로드를 제한으로 설정합니다.
            DEXT5UPLOAD.SetAllowOrLimitExtension(0, 'gif,jpg', 'upload1');    
        }   
</script>	

<!-- ..... 생략 ..... -->

<div style="width:800px;height:200px">  
        <script type="text/javascript">
            new Dext5Upload("upload1");
        </script>
</div>
