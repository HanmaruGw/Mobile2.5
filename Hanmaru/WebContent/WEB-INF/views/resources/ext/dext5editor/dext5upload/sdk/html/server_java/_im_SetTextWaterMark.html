DEXT5 Upload :: SetTextWaterMark

void SetTextWaterMark
(String strSourceFile, TextWaterMark pTextWaterMark, String strVAlignFrom, int nVMargin, String strHAlignFrom, int nHMargin, int nTransparency, int nDegree)

파일 업로드 시 이미지 파일인 경우 텍스트 워터마크를 적용할 경우 사용됩니다.
이벤트 UploadCompleteBeforeEvent, UploadCompleteEvent 에서 사용이 가능합니다.

return value

없음.

parameters

strSourceFile  이미지 파일의 물리적 경로를 의미합니다.
pTextWaterMark  워터마크 처리할 텍스트의 정보를 담은 Object를 의미합니다.
strVAlignFrom  워터마크가 위치할 수직정렬의 값을 의미합니다. TOP,MIDDLE,BOTTOM
nVMargin  워터마크가 위치할 수직 Margin 값을 의미합니다.
strHAlignFrom  워터마크가 위치할 수평정렬의 값을 의미합니다. LEFT,CENTER,RIGHT
nHMargin  워터마크가 위치할 수평 Margin 값을 의미합니다.
nTransparency  워터마크의 투명도를 의미합니다.
nDegree  워터마크가 표시될 각도를 의미합니다.

remarks

없음.
