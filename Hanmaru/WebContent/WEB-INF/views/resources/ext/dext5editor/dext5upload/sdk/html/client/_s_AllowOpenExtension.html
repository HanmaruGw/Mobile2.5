DEXT5 Upload :: AllowOpenExtension

XML : <allow_open_extension>.....</allow_open_extension>, Config : config.AllowOpenExtension

Config 설정이 우선 적용됩니다.
(웹표준 모드) 열기 동작시 popup으로 열기할 확장자를 설정합니다.

remarks

기본값은 빈값으로 전체 확장자를 사용이고,
확장자명을 콤마(,)로 구분하여 설정시 설정된 확장자의 파일만 열기가 되고, 나머지 파일은 다운로드 됩니다.

기본적으로 다음 설정값이 설정되도록 가이드 됩니다.
bmp,gif,jpeg,jpg,png,tif,tiff,swf,htm,html,xml,txt

sample code

열기 동작시 popup으로 열기할 확장자를 "bmp,gif,jpeg,jpg,png,tif,tiff,swf,htm,html,xml,txt"로 설정합니다.

XML

.....
<setting>
    .....
    <allow_open_extension>bmp,gif,jpeg,jpg,png,tif,tiff,swf,htm,html,xml,txt</allow_open_extension>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        DEXT5UPLOAD.config.AllowOpenExtension = 'bmp,gif,jpeg,jpg,png,tif,tiff,swf,htm,html,xml,txt';

        new Dext5Upload("upload1");
    </script>
</div>
