DEXT5 Editor :: PasteToImage

XML : <paste_to_image>.....</paste_to_image>
Config : config.PasteToImage

Config 설정이 우선 적용됩니다.
에디터 마우스 우클릭 시 이미지로 붙여넣기 항목을 설정합니다.

remarks

기본값은 "1" 이고 기본 붙여넣기에 이미지로 붙여넣기를 설정, "0" 으로 설정 시 우클릭 이미지로 붙여넣기 항목을 설정합니다.

sample code

에디터 마우스 우클릭 시 이미지로 붙여넣기 항목으로 설정합니다.

XML

.....
<setting>
    .....
    <paste_to_image>0</paste_to_image>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">  
    <script type="text/javascript">

        DEXT5.config.PasteToImage = "0";

        new Dext5editor("editor1");

    </script>       
</div>
