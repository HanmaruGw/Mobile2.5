DEXT5 Editor :: dext5DecodeMime

string dext5DecodeMime(mimeData)

mimeData를 decode합니다.

return value

정상 완료 시 디코딩 소스를 오류 시 "0"을 리턴합니다.

parameters

mimeData  디코딩할 mimeData입니다.

remarks

에디터 설정파일(dext5editor/config/dext_editor.xml)의 mime_use 설정값이 "1" 이어야 합니다.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
<script type="text/javascript">

    function decodeMime() {
        // id가 editor1인 에디터 디자인 영역의 html소스를 mime형태로 변환합니다.
        var html = DEXT5.getBodyValue('editor1');
        var mimedata = DEXT5.dext5EncodeMime(html, 0);
            
        // 변환된 mimeData를 다시 decode합니다.
        var decodeHtml = DEXT5.dext5DecodeMime(mimedata);
    }

</script>
