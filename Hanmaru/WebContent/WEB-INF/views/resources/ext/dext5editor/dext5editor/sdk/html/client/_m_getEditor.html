DEXT5 Editor :: getEditor

object getEditor(editor_id)

생성된 에디터를 가져옵니다.
다중 에디터를 생성하여 작업할 경우 유용하게 쓰일 수 있습니다.

return value

에디터 객체를 리턴합니다.

parameters

editor_id  가져올 에디터의 id를 의미합니다.

remarks

없음.

sample code
