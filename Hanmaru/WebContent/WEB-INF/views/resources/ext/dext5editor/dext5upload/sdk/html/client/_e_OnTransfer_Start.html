DEXT5 Upload :: OnTransfer_Start

boolean DEXT5UPLOAD_OnTransfer_Start(uploadID)

파일 전송 시작 전 발생합니다.

return value

boolean  true일 경우 전송 시작하고, false일 경우 전송하지 않습니다.

parameters

uploadID  전송이 시작된 업로드의 id를 의미합니다.

remarks

없음.

sample code

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>	

<script type="text/javascript">
        function DEXT5UPLOAD_OnTransfer_Start(uploadID) {
            // 업로드 시작 후 처리할 내용

            return true or false;
        }
</script>
