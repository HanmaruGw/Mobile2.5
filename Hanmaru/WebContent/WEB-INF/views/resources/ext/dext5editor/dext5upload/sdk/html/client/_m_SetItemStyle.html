DEXT5 Upload :: SetItemStyle

void SetItemStyle(itemIndex, paramObj, uploadID)

업로드에 추가된 파일리스트의 파일명에 스타일을 변경합니다.

return value

없음.

parameters

itemIndex  0부터 시작하는 파일index, -1일 경우 전체파일을 의미합니다.
paramObj  파일명 부분의 변경할 스타일을 의미합니다.
uploadID  파일명의 스타일을 변경할 업로드의 id를 의미합니다.

remarks

FileNameColor  파일명 색상
Bold  파일명 굵기
Italic  이탤릭체
UnderLine  파일명 밑줄

sample code

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>	
<script type="text/javascript">
            DEXT5UPLOAD.SetItemStyle(-1, { FileNameColor: '#a4a4a4', Bold: true, Italic: true, UnderLine: true }, "upload1");
</script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
        <script type="text/javascript">
            new Dext5Upload("upload1");
        </script>
</div>
