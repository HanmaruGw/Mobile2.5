DEXT5 Upload :: SetViewerCssPath

upload.SetViewerCssPath

웹표준 모드의 뷰어사용시 뷰어에 사용되는 dext5upload/css/dext5viewer.min.css 파일의 경로를 설정합니다.
일반적으로 dext5upload/handler/dext5viewer.aspx를 다른 경로에 사용될 경우 설정합니다.
이 경우 dext5viewer.aspx 페이지에서 dext5viewer.min.css의 경로가 접근 가능하도록 설정해야 합니다.

remarks

기본값 : ../css/dext5viewer.min.css ( dext5upload/css/dext5viewer.min.css )

sample code

//dext5viewer.aspx에서 설정

Raonwiz.Dext5.UploadHandler upload = new Raonwiz.Dext5.UploadHandler();

upload.SetViewerCssPath = "../css/dext5viewer.min.css";

upload.Viewer(this.Context);
