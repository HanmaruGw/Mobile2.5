DEXT5 Upload :: TransferEx

void TransferEx(uploadID)

파일추가 완료 후 파일을 전송합니다.

return value

없음.

parameters

uploadID  전송 시작을 동작시킬 업로드의 id를 의미합니다.

remarks

파일추가 완료 후 발생하는 함수 DEXT5UPLOAD_AfterAddItemEndTime(uploadID) 에서 사용합니다.

sample code

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>	
<script type="text/javascript">
        function DEXT5UPLOAD_AfterAddItemEndTime(uploadID) {
            // 파일 추가후 처리할 내용
            DEXT5UPLOAD.TransferEx(uploadID);
        }
</script>
