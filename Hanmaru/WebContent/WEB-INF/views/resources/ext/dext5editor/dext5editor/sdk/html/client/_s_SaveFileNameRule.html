DEXT5 Editor :: SaveFileNameRule

XML : <save_filename_rule>.....</save_filename_rule>, Config : config.SaveFileNameRule

Config 설정이 우선 적용됩니다.
웹에서 등록된 파일의 이름을 지정하는 규칙입니다.

remarks

기본값은 "GUID" 입니다.
값 지정은 "GUID" 또는 "DATETIME" 를 입력합니다.
"DATETIME" 의 경우 파일이름이 년도(4자리)월(2자리)일(2자리)시간(2자리)시(2자리)초(2자리)밀리초(2자리) 형태로 설정됩니다.

sample code

웹에서 등록된 파일의 이름을
년도(4자리)월(2자리)일(2자리)시간(2자리)시(2자리)초(2자리)밀리초(2자리) 형태로 설정함니다.

XML

.....
<uploader_setting>
    .....
    <save_filename_rule>DATETIME</save_filename_rule>
    .....
</uploader_setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">

    </script>       
</div>
