DEXT5 Upload :: BeforeAddItem

boolean DEXT5UPLOAD_BeforeAddItem(uploadID, strFileName, nFileSize, nAddItemIndex, strFileLocalPath, objFile)

파일 추가 전 발생합니다.

return value

boolean  true일 경우 파일을 첨부하고, false일 경우 첨부하지 않습니다.

parameters

uploadID  파일 추가 요청한 업로드의 id를 의미합니다.
strFileName  추가될 파일명을 의미합니다.
nFileSize  파일 사이즈를 의미합니다.
nAddItemIndex  1부터 시작되는 파일의 순서정보를 의미합니다.
strFileLocalPath  파일 로컬 경로 정보를 의미합니다.
objFile  추가될 파일객체를 의미합니다.(html5모드)

remarks

DEXT5 Upload 설정 config.xml에서 use_add_event 값이 "1" 일 경우 발생합니다.

sample code

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>	

<script type="text/javascript">
        function DEXT5UPLOAD_BeforeAddItem(uploadID, strFileName, nFileSize, nAddItemIndex, strFileLocalPath, objFile) {
            // 파일 추가 전 처리할 내용

               return true or false;
        }
</script>
