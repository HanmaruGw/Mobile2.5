DEXT5 Editor :: TableLineStyleList

XML : <line_style>.....</line_style>
Config : config.TableLineStyleList

Config 설정이 우선 적용됩니다.
테이블에 선모양 기본 스타일을 설정합니다.

remarks

기본값은 solid이고, solid, dotted, dashed, double, groove, ridge, inset, outset, none 중 하나의 값을 설정합니다.

sample code

테이블에 선모양 기본 스타일을 dotted로 설정합니다.

XML

.....
<setting>
    .....
    <table_property>
        .....
        <line_style>dotted</line_style>
        .....
    </table_property>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>
