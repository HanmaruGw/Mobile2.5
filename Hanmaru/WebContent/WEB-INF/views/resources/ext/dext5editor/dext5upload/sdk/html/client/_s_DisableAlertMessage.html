DEXT5 Upload :: DisableAlertMessage

XML :
<disable_alert_message>
    <disable_delete_confirm>.....</disable_delete_confirm>
    <duplication>.....</duplication>
    <delete_unchosen>.....</delete_unchosen>
    <download_unchosen>.....</download_unchosen>
    <open_unchosen>.....</open_unchosen>
</disable_alert_message>
Config :
config.DisableAlertMessage = {
    DisableDeleteConfirm: '',
    Duplication: '',
    DeleteUnchosen: '',
    DownloadUnchosen: '',
    OpenUnchosen: '',
}

Config 설정이 우선 적용됩니다.
업로드 모든 confirm창을 표시하거나 숨기는 기능을 설정합니다.

remarks

XML:disable_delete_confirm, Config:DisableDeleteConfirm  기본값은 "0" 이고 "모든 confirm창 활성", "1"로 설정시 "모든 confirm창 비활성"입니다.
"2"로 설정시 "선택 항목제거만 confirm창 비활성"이고,
"3"로 설정시 "전체 항목제거만 confirm창 비활성"입니다.

XML:duplication, Config:Duplication  "파일 추가 중복 시" alert창 비활성합니다.
기본값은 "1" 이고, "0"으로 설정시 파일 추가 중복 시 alert창 활성합니다.

XML:delete_unchosen, Config:DeleteUnchosen  "파일 삭제 시" 선택된 파일이 없을 경우 alert창 비활성합니다.
기본값은 "1" 이고, "0"으로 설정시 파일 삭제시 선택된 파일이 없을 경우 alert창 활성합니다.

XML:download_unchosen, Config:DownloadUnchosen  "파일 다운로드 시" 선택된 파일이 없을 경우 alert창 비활성합니다.
기본값은 "1" 이고, "0"으로 설정시 파일 삭제시 선택된 파일이 없을 경우 alert창 활성합니다.

XML:open_unchosen, Config:OpenUnchosen  "파일 열기 시" 선택된 파일이 없을 경우 alert창 비활성합니다.
기본값은 "1" 이고, "0"으로 설정시 파일 삭제시 선택된 파일이 없을 경우 alert창 활성합니다.

sample code

모든 confirm창을 표시하거나 숨기는 기능을 설정합니다.
파일 삭제에 대한 모든 confirm창 비활성으로 설정합니다.
파일 추가 중복 시 alert창 활성으로 설정합니다.
파일 삭제 시 alert창 활성으로 설정합니다.
파일 다운로드 시 alert창 활성으로 설정합니다.
파일 열기 시 alert창 활성으로 설정합니다.
