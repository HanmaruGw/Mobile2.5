DEXT5 Upload :: AddLocalFileObject

void AddLocalFileObject(fileObject, fileMarkValue, uploadId)

File 태그와 업로드 연동이 필요한 경우 파일을 추가 하고 전송할 수 있습니다.

return value

없음.

parameters

fileObject  첨부할 파일태그를 의미합니다.
fileMarkValue  첨부하는 파일의 mark 값이 필요한 경우 값을 의미합니다.
전송 완료 후 각 파일의 입력하신 mark 값이 리턴 됩니다.
uploadID  첨부하려는 업로드의 id값을 의미합니다.

remarks

없음.
