DEXT5 Editor :: DefaultLineHeight

XML : <line_height>.....</line_height>, Config : config.DefaultLineHeight
XML : <line_height mode="....."></line_height>, Config : config.LineHeightMode

Config 설정이 우선 적용됩니다.
에디터 로드 시 사용할 기본 줄간격과 줄간격 Mode(% or 배수)를 설정합니다.

remarks

XML:line_height,Config:DefaultLineHeight  기본 줄간격은 "1.2" 입니다.
XML:mode,Config:LineHeightMode  기본 Mode는 "2" 배율, "1"로 설정시 %로 설정됩니다.

sample code

에디터 기본 줄간격을 1.5(배수)로 설정합니다.

XML

.....
<setting>
    .....
    <line_height mode="2">1.5</line_height>
    .....
</setting>
.....
