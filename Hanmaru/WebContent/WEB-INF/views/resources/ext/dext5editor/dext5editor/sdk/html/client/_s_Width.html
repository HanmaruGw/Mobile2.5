DEXT5 Editor :: Width

XML : <width>.....</width>, Config : config.Width

Config 설정이 우선 적용됩니다.
에디터의 너비값을 설정합니다.

remarks

기본값은 "900px" 으로 설정됩니다.

sample code

에디터의 너비값을 750px 으로 설정합니다.

XML

.....
<setting>
    .....
    <width>750px</width>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
     
<!-- ..... 생략 ..... -->
     
<div style="width:900px;height:550px">  
    <script type="text/javascript">

        DEXT5.config.Width = "750px";
    
        new Dext5editor("editor1");

    </script>       
</div>
