DEXT5 Upload :: BeforeFileDownload

boolean DEXT5UPLOAD_BeforeFileDownload(uploadID, nWebFile, strItemKey, strItemOrgName, strItemUrlOrPath, largeFiles, isLastEvent)

다운로드 전 발생합니다.

return value

boolean  true일 경우 파일을 다운로드하고, false일 경우 다운로드하지 않습니다.

parameters

uploadID  다운로드 요청한 업로드의 id를 의미합니다.
nWebFile  Web파일과 Local 파일의 구분 값을 의미합니다. (0:local, 1:web)
strItemKey  파일의 Unique Key를 의미합니다.
strItemOrgName  파일의 원본 파일명을 의미합니다.
strItemUrlOrPath  파일의 저장위치를 의미합니다.
largeFiles  파일 대용량를 의미합니다.
isLastEvent  개별 다운로드와 전체 다운로드 구별을 의미합니다.

remarks

DEXT5 Upload 설정 config.xml에서 use_download_event 값이 1일 경우 발생합니다.

sample code

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>	

<script type="text/javascript">
        function DEXT5UPLOAD_BeforeFileDownload(uploadID, nWebFile, strItemKey, strItemOrgName, strItemUrlOrPath, largeFiles, isLastEvent) {
            // 다운로드 전 처리할 내용

            return true or false;
        }
</script>
