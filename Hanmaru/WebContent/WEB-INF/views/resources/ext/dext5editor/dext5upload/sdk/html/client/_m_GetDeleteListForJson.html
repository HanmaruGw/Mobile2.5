DEXT5 Upload :: GetDeleteListForJson

object GetDeleteListForJson(uploadID)

전송완료 후 업로드 되었던 파일 중 삭제된 파일 리스트를 json 형태로 리턴합니다.

return value

object  전송완료 후 업로드 되었던 파일 중 삭제된 파일 리스트를 의미합니다.

parameters

uploadID  파일 리스트를 가져올 업로드의 id를 의미합니다.

remarks

없음.
