DEXT5 Upload :: UseDropzone

XML : <use_dropzone>.....</use_dropzone>, Config : config.UseDropzone

Config 설정이 우선 적용됩니다.
Drop Zone 이벤트 함수 사용을 설정합니다.

remarks

DropZone설정값은 다음과 같습니다.
Xml파일 - use_dropzone
프로퍼티 - DEXT5UPLOAD.config.UseDropzone

기본값은 "0" 미사용이고, "1"로 설정시 Drop Zone으로 만들려는 객체의 클래스명이 "dext5dropzone"을 포함하고 있다면 Drop Zone으로 만듭니다.
"1"로 설정 시 Drop Zone에 파일을 Drag&Drop하면 DEXT5UPLOAD_DropZoneAddItem(fileObject, filePath, objectId) 함수가 호출 됩니다.

sample code

Drop Zone 이벤트 함수 사용으로 설정합니다.

XML

.....
<setting>
    .....
    <use_dropzone>1</use_dropzone>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        DEXT5UPLOAD.config.UseDropzone = "1";

        new Dext5Upload("upload1");
    </script>
</div>

Event Script & Event Area

<script type="text/javascript">
        // DropZone에 Drag&Drop 시 다음의 이벤트가 발생합니다.
        function DEXT5UPLOAD_DropZoneAddItem(fileObject, filePath, objectId) {

            // DropZone에 Drag&Drop한 파일을 업로드리스트에 추가하는 API입니다.
            DEXT5UPLOAD.AddDropZoneFile(fileObject, filePath, "upload1");
        }
</script>
