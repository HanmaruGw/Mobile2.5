DEXT5 Upload :: FolderNameRule

XML : <folder_name_rule>.....</folder_name_rule>, Config : config.FolderNameRule

Config 설정이 우선 적용됩니다.
dext5handler에서 설정된 폴더 하위들의 저장 체계를 설정합니다.

remarks

일반 폴더 또는 년월일폴더를 사용할 수 있습니다.

일반폴더는 "사용자 설정값" 형태로 사용할 수 있습니다.
년월일폴더는
사용하지 않음 => "/",
년도 => "YYYY/",
년도/월 => "YYYY/MM/",
년도/월/일 => "YYYY/MM/DD/" 중 하나의 값을 사용합니다.

폴더가 존재하지 않으면 자동으로 생성되며, 서버의 날짜가 기준이 됩니다.

sample code

업로드 할 폴더를 uploadfolder 로 설정합니다.

XML

.....
<uploader_setting>
    .....
    <folder_name_rule>uploadfolder</folder_name_rule>
    .....
</uploader_setting>
.....

Config

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        DEXT5UPLOAD.config.FolderNameRule = 'uploadfolder';

        new Dext5Upload("upload1");
    </script>
</div>
