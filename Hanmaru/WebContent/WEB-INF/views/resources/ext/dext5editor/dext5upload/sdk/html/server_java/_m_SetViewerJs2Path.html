DEXT5 Upload :: SetViewerJs2Path

void SetViewerJs2Path(String viewerJs2Path)

웹표준 모드의 뷰어사용시 뷰어에 사용되는 dext5upload/js/dext5upload.base64.js 파일의 경로를 설정합니다.
일반적으로 dext5upload/handler/dext5viewer.jsp를 다른 경로에 사용될 경우 설정합니다.
이 경우 dext5viewer.jsp 페이지에서 dext5upload.base64.js의 경로가 접근 가능하도록 설정해야 합니다.

return value

없음.

parameters

viewerJs2Path  웹표준 모드의 뷰어사용시 뷰어에 사용되는 dext5upload/js/dext5upload.base64.js 파일의 경로를 의미합니다.

remarks

기본값 : ../js/dext5upload.base64.js ( dext5upload/js/dext5upload.base64.js )

sample code

Raonwiz.Dext5.UploadHandler upload = new Raonwiz.Dext5.UploadHandler();

upload.SetViewerJs2Path("../js/dext5upload.base64.js");

String result = upload.Process(request, response, event);
