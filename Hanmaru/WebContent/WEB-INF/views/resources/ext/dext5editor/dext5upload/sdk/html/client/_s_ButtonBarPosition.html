DEXT5 Upload :: ButtonBarPosition

XML : <button_bar_position>.....</button_bar_position>, Config : config.ButtonBarPosition

Config 설정이 우선 적용됩니다.
업로드가 편집/보기모드일 때 버튼바 영역 위치를 설정합니다.

remarks

기본값은 "bottom"(하단) 이고, "top"(상단) 설정시 전송영역 맨 위에 버튼바가 설정됩니다.

sample code

업로드 버튼바를 상단에 위치합니다.

XML

.....
<setting>
    .....
    <button_bar_position>top</button_bar_position>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        DEXT5UPLOAD.config.ButtonBarPosition = 'top';

        new Dext5Upload("upload1");
    </script>
</div>
