DEXT5 Upload :: GetExifEntityData

ExifEntity GetExifEntityData(string pSourceFileFullPath)

파일 업로드 시 이미지 파일인 경우 이미지의 EXIF 정보를 추출하려는 경우 사용됩니다. (Exif standard 2.2, JEITA CP-2451)
이벤트 UploadCompleteBeforeEvent, UploadCompleteEvent 에서 사용이 가능합니다.

return value

Raonwiz.Dext5.Common.Exif.ExifEntity  이미지의 EXIF 정보를 의미합니다.

parameters

pSourceFileFullPath  이미지 파일의 물리적 경로를 의미합니다.

remarks

없음.

sample code

string newFileLocation = parameterEntity.NewFileLocation;
using (Raonwiz.Dext5.Common.Dext5Image dextImage = new Raonwiz.Dext5.Common.Dext5Image())
{
    try
    {
        string sourceFileFullPath = newFileLocation;
        
        Raonwiz.Dext5.Common.Exif.ExifEntity _exif = dextImage.GetExifEntityData(sourceFileFullPath);

    }
    catch (Exception ex) { }
}
