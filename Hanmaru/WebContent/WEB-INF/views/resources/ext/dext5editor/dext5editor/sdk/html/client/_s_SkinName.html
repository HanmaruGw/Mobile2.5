DEXT5 Editor :: SkinName

XML : <skinname>.....</skinname>, Config : config.SkinName

Config 설정이 우선 적용됩니다.
에디터의 스킨 색상을 설정합니다.

remarks

기본값은 "blue" 색상으로 설정됩니다.

스킨 색상은
blue, brown, darkgray, gold, gray, green, orange, pink, purple, red, silver, yellow, random
12가지 입니다.

sample code

에디터의 스킨을 gray 로 설정합니다.

XML

.....
<setting>
    .....
    <skinname>gray</skinname>
    .....
</setting>
.....
