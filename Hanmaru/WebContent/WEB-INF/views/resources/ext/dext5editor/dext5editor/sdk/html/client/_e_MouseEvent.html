DEXT5 Editor :: dext_mouse_event

void dext_mouse_event(eventType, event, currentElement)

에디터 작성영역에서 마우스이벤트가 발생했을 경우 발생합니다.

return value

없음.

parameters

eventType  발생한 마우스이벤트의 종류입니다.
event  발생한 마우스이벤트입니다.
currentElement  마우스이벤트가 발생한 요소입니다.

remarks

에디터의 설정파일(dext5editor/config/dext_editor.xml)의 return_event 설정의 mouse_event 속성값이 1인 경우에 이벤트가 발생합니다.
마우스이벤트 종류에 따른 eventType값은 다음과 같습니다.
click이벤트 - eventType 값은 "click"
contextmenu이벤트 - eventType 값은 "contextmenu"
dblclick이벤트 - eventType 값은 "dblclick"
mousedown이벤트 - eventType 값은 "mousedown"
mouseup이벤트 - eventType 값은 "mouseup"
mousemove이벤트 - eventType 값은 "mousemove"
