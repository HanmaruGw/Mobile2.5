DEXT5 Upload :: SetDownloadCheckFileExtension

void SetDownloadCheckFileExtension(int iAllowOrLimit, String sFileExtensionList)

다운로드시 서버에서 보안을 위하여 확장자 체크를 합니다. (부모 경로 접근을 이용한 서버파일 다운로드 방지 등)
아래 부분을 적용하시면, 설정한 값으로 서버에서 확장자 체크가 이루어집니다.

return value

없음.

parameters

iAllowOrLimit  0: 제한으로 설정, 1: 허용으로 설정
sFileExtensionList  확장자 목록 ex) jpg,exe (구분자,)

remarks

기본값 : 확장자 체크 안함.

sample code

Raonwiz.Dext5.UploadHandler upload = new Raonwiz.Dext5.UploadHandler();

upload.SetDownloadCheckFileExtension(0, "exe,aspx,jsp");

String result = upload.Process(request, response, event);
