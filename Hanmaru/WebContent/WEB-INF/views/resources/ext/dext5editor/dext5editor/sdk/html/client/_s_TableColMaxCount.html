DEXT5 Editor :: TableColMaxCount

XML : <table_property col_max_count="....."></table_property>
Config : config.TableColMaxCount

Config 설정이 우선 적용됩니다.
표 삽입 시 열의 최대값을 설정합니다.

remarks

기본 최대값은 "13" 입니다.

sample code

표 삽입 시 열의 최대값을 20으로 설정합니다.

XML

.....
<setting>
    .....
    <table_property col_max_count="20">
        .....
    </table_property>
    .....
</setting>
.....
