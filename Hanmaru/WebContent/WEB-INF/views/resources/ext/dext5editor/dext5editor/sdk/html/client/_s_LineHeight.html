DEXT5 Editor :: LineHeight

XML : <line_height>.....</line_height>, Config : config.LineHeightList

Config 설정이 우선 적용됩니다.
에디터 줄간격 메뉴에 노출할 줄간격 리스트를 설정합니다.

remarks

XML:line_height  필요 시 <line>.....</line> 항목을 추가하여 설정합니다.
Config:LineHeightList  필요 시 DEXT5.config.LineHeightList = "LineHeight값";에 콤마(,)로 구분하여 설정합니다.

sample code

에디터 줄간격 메뉴에 노출할 줄간격을 1, 1.2 로 설정합니다.

XML

.....
<line_height>
    <line>1</line>
    <line>1.2</line>
</line_height>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">  
    <script type="text/javascript">

        DEXT5.config.LineHeightList = "1,1.2";

        new Dext5editor("editor1");

    </script>       
</div>
