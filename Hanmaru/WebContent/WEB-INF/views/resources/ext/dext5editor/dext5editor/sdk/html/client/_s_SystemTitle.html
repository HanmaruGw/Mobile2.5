DEXT5 Editor :: SystemTitle

XML : <system_title>.....</system_title>, Config : config.SystemTitle

Config 설정이 우선 적용됩니다.
다이얼로그창 상단 문구 변경 시 설정합니다.

remarks

기본값은 "DEXT5" 이고, "0" 으로 설정 시 사용하지 않음, "사용자 입력값" 으로 설정합니다.

sample code

다이얼로그창 상단 문구를 Custom Title 로 설정합니다.

XML

.....
<setting>
    .....
    <system_title>Custom Title</system_title>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
     
<!-- ..... 생략 ..... -->
     
<div style="width:900px;height:550px">  
    <script type="text/javascript">

        DEXT5.config.SystemTitle = "Custom Title";
    
        new Dext5editor("editor1");

    </script>       
</div>
