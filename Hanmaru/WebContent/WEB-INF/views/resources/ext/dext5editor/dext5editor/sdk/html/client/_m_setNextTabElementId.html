DEXT5 Editor :: setNextTabElementId

void setNextTabElementId(strId, editor_id)

포커스를 줄 객체 아이디를 설정 하고 tab키를 입력하면 마지막 포커스가 설정한 아이디로 이동합니다.

return value

없음.

parameters

strId   포커스 줄 객체 아이디를 의미합니다.
editor_id  접근할 에디터의 id를 의미합니다.

remarks

에디터에서 tab 사용으로 마지막 포커스가 설정한 아이디로 이동하고 싶을 때 유용합니다.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
<script type="text/javascript">

    function setNextTabElementId(){
        // 에디터에서 tab키 사용으로 마지막 포커스를 설정한 ID로 이동합니다.
        DEXT5.setNextTabElementId('title');
    }

</script>	

<input type="text" name="title" id="title" value="" />
