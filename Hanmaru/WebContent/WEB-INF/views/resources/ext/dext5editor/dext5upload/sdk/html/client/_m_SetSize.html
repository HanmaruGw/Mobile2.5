DEXT5 Upload :: SetSize

void SetSize(uploadID)

업로드 생성 후 업로드의 크기를 변경할 수 있습니다.

return value

없음.

parameters

width   업로드의 가로 값을 의미 합니다. (% 또는 px 모두 가능)
height  업로드의 세로 값을 의미 합니다. (px만 가능)
uploadID  실행할 업로드의 id를 의미합니다. (단일 업로드일 경우 생략 가능)

remarks

업로드를 1개만 생성했을 경우 uploadID를 입력하지 않아도 업로드 크기를 설정할 수 있습니다.
    예) id가 upload1인 업로드를 생성했을 경우
     DEXT5UPLOAD.SetSize("700", "500", "upload1"), DEXT5UPLOAD.SetSize("700", "500")
    모두 upload1업로드의 크기를 설정합니다.

업로드를 2개 이상 생성했을 경우 uploadID를 반드시 입력해야 합니다.
    예) id가 upload1, upload2인 업로드를 생성했을 경우
     DEXT5UPLOAD.SetSize("700", "500", "upload1") upload1 업로드의 크기를 설정합니다.
     DEXT5UPLOAD.SetSize("700", "500", "upload2") upload2 업로드의 크기를 설정합니다.

sample code

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>	
<script type="text/javascript">          
        function setSize() {
            DEXT5UPLOAD.SetSize('700','500','upload1');
        }
</script>
