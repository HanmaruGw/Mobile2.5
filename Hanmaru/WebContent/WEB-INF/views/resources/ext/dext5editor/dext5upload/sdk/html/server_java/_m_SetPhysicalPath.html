DEXT5 Upload :: SetPhysicalPath

void SetPhysicalPath(String physicalPath)

실제 업로드 할 기본경로를 물리적 경로로 설정합니다.
폴더명 제일 뒤에 .과 공백이 있다면 제거하시고 설정해 주세요.(운영체제에서 지원되지 않는 문자열입니다.)

return value

없음.

parameters

physicalPath  실제 업로드 할 물리적 경로를 의미합니다.

remarks

기본값은 ""이고, SetPhysicalPath 또는 SetVirtualPath 중에 반드시 하나는 설정해야 업로드가 가능합니다.

sample code

Raonwiz.Dext5.UploadHandler upload = new Raonwiz.Dext5.UploadHandler();

upload.SetPhysicalPath("D:\\dext5upload\\dext5uploaddata");

String result = upload.Process(request, response, event);
