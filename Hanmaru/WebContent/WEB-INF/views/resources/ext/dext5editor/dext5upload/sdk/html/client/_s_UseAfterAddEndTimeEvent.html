DEXT5 Upload :: UseAfterAddEndTimeEvent

XML : <use_after_add_end_time_event>.....</use_after_add_end_time_event>
Config : config.UseAfterAddEndTimeEvent

Config 설정이 우선 적용됩니다.
업로드 리스트 컨트롤에서 모든 파일추가 완료 후 이벤트 함수 사용을 설정합니다.

remarks

기본값은 "1" 사용이고, "0"으로 설정시 이벤트 미사용합니다.
"1"으로 설정 후 파일 추가 완료 시 DEXT5UPLOAD_AfterAddItemEndTime(uploadID) 함수가 호출 됩니다.

sample code

모든 파일 추가 완료 후 이벤트 함수 미사용으로 설정합니다.

.....
<setting>
    .....
    <use_after_add_end_time_event>0</use_after_add_end_time_event>
    .....
</setting>
.....
