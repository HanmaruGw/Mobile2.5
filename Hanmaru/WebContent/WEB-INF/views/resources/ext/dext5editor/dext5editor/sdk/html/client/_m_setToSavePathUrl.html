DEXT5 Editor :: setToSavePathUrl

void setToSavePathUrl(value, editor_id)

에디터의 ToSavePathUrl(서버에 연결 된 저장소의 URL 경로)값을 설정합니다.
(기본값은 dext5editordata로 설정되어 있습니다.)

return value

없음.

parameters

value  에디터에 설정할 ToSavePathUrl 값을 의미합니다.
editor_id  설정할 에디터의 id를 의미합니다.

remarks

에디터를 1개만 생성했을 경우 parameter를 입력하지 않아도 ToSavePathUrl 값을 설정할 수 있습니다.
예) id가 editor1인 에디터를 생성했을 경우
     DEXT5.setToSavePathUrl('/dext5editordata', 'editor1'), DEXT5.setToSavePathUrl('/dext5editordata')
     모두 editor1 에디터의 ToSavePathUrl을 dext5editordata로 설정합니다.

에디터를 2개 이상 생성했을 경우 parameter를 반드시 입력해야 합니다.
예) id가 editor1, editor2인 에디터를 생성했을 경우
     DEXT5.setToSavePathUrl('/dext5editordata', 'editor1')   editor1 에디터의 ToSavePathUrl을 dext5editordata로 설정합니다.
     DEXT5.setToSavePathUrl('/dext5editordata', 'editor2')   editor2 에디터의 ToSavePathUrl을 dext5editordata로 설정합니다.

    }

</script>
