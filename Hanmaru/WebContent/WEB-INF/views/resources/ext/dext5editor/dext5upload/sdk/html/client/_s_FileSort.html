DEXT5 Upload :: FileSort

XML : <file_sort>.....</file_sort>, Config : config.UseFileSort
XML : <file_sort sort_field="....."></file_sort>, Config : config.FileSortField
XML : <file_sort sort_ascdesc="....."></file_sort>, Config : config.FileSortAscDesc
XML : <file_sort auto_sort="....."></file_sort>, Config : config.AutoSort

Config 설정이 우선 적용됩니다.
파일 정렬 기능을 설정합니다.

remarks

XML:file_sort, Config:UseFileSort  "0"이면 파일 정렬 기능 사용 안함이고, "1"으로 설정시 파일 정렬 기능 사용입니다.
XML:sort_field, Config:FileSortField  정렬 대상 필드 값
"0"이면 "Name 필드 정렬"이고, "1"로 설정시 "Size 필드 정렬"입니다.
XML:sort_ascdesc, Config:FileSortAscDesc  정렬 방법
"0"이면 "오름차순", "1"이면 "내림차순" 입니다.
XML:auto_sort, Config:AutoSort  자동 정렬 수행 방법
"0"이면 "자동 정렬 사용 안함"이고,
"1"으로 설정시 "전체 다시 정렬" (새로운 파일이 첨부되었을 경우 설정한 정렬 옵션으로 전체 다시 정렬),
"2"으로 설정시 "새로 추가 된 파일만 정렬" (새로 추가되는 파일만을 설정한 정렬 옵션으로 정렬) 입니다.

sample code

정렬기능 사용하고, Name 필드로 정렬 설정
오름차순 정렬 설정하고, 자동 정렬은 사용하지 않도록 설정
