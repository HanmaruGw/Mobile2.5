DEXT5 Upload :: Width

XML : <width>.....</width>, Config : config.Width

Config 설정이 우선 적용됩니다.
업로드의 넓이를 설정합니다.

remarks

기본값은 "800px" 입니다.
넓이는 "px", "%" 모두 가능합니다.

sample code

업로드의 너비값을 500px 으로 설정합니다.

XML

.....
<setting>
    .....
    <width>500px</width>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        DEXT5UPLOAD.config.Width = '500px';

        new Dext5Upload("upload1");
    </script>
</div>
