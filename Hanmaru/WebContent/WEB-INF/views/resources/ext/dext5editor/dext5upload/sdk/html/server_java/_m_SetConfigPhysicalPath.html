DEXT5 Upload :: SetConfigPhysicalPath

void SetConfigPhysicalPath(String configPhysicalPath)

환경설정파일을 서버에서 관리하고, 서버에 저장된 환경설정파일을 사용할 경우 사용합니다.

return value

없음.

parameters

configPhysicalPath  서버에 저장된 환경설정파일의 경로를 의미합니다.

remarks

기본값은 사용안합

sample code

Client

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        // Full URL은 DEXT5 Upload Handler경로로 지정하며
        // f 파라미터의 값은 서버에서 처리할 XML 파일명을 지정합니다.
        DEXT5UPLOAD.config.InitServerXml = "http://localhost/handler/dext5handler.jsp?f=dext5upload.config.xml";
        new Dext5Upload("upload1");
    </script>
</div>

Server

Raonwiz.Dext5.UploadHandler upload = new Raonwiz.Dext5.UploadHandler();
upload.SetConfigPhysicalPath("D:\\config");
String result = upload.Process(request, response, event);
