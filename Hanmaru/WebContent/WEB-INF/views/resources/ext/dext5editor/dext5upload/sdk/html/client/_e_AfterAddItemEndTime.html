DEXT5 Upload :: AfterAddItemEndTime

void DEXT5UPLOAD_AfterAddItemEndTime(uploadID)

파일 추가 완료 후 발생합니다.

return value

없음.

parameters

uploadID  파일 추가 완료된 업로드의 id를 의미합니다.

remarks

DEXT5 Upload 설정 config.xml에서 use_after_add_end_time_event 값이 1일 경우 발생합니다.

        }
</script>
