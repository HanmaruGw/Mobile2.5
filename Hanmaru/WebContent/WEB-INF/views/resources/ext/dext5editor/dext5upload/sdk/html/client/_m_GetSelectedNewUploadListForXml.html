DEXT5 Upload :: GetSelectedNewUploadListForXml

object GetSelectedNewUploadListForXml(uploadID)

선택된 파일중 사용자가 추가한 파일 리스트를 xml 형태로 리턴합니다.

return value

object  선택된 파일중 사용자가 추가한 파일 리스트를 의미합니다.

parameters

uploadID  파일 리스트를 가져올 업로드의 id를 의미합니다.

remarks

없음.

sample code

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>	
<script type="text/javascript">
        var file_list = DEXT5UPLOAD.GetSelectedNewUploadListForXml("upload1");

        // 아래의 xml구조를 참고하셔서 xml을 파싱하신 후 파일정보를 추출하시면 됩니다.
        //<files>
        //    <file>
        //        <originalname> // 원본파일명
        //        <uploadname> // 업로드 된 파일명

        //        <uploadpath> // 업로드 경로

        //        <logicalpath> // 폴더구조 업로드 시 업로드된 파일의 폴더경로
        //        <extension> // 확장자
        //        <localpath> // 로컬경로
        //        <mark> // markValue
        //        <responsecustomvalue> // 서버로부터 받은 customValue
        //        <order> // 업로드리스트에서의 순서
        //    </file>
        //    <file>
