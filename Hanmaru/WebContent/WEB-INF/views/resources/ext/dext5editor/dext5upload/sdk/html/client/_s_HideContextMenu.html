DEXT5 Upload :: HideContextMenu

XML : <hide_context_menu>.....</hide_context_menu>, Config : config.HideContextMenu

Config 설정이 우선 적용됩니다.
업로드 context menu(우 클릭 메뉴) 사용 여부를 설정합니다.

remarks

기본값은 "0" 사용이고, "1"로 설정시 context menu(우 클릭 메뉴) 미사용을 설정합니다.

sample code

context menu(우 클릭 메뉴) 미사용으로 설정합니다.

XML

.....
<setting>
    .....
    <hide_context_menu>1</hide_context_menu>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>
