DEXT5 Editor :: UserFieldID,

DEXT5 Editor :: UserFieldValue

config.UserFieldID, config.UserFieldValue

파일 업로드에 사용되는 User 태그의 name과 value를 설정합니다.

remarks

User태그의 name은 UserFieldID로 설정합니다.
기본값은 "Userdata" 로 설정됩니다.

User태그의 value는 UserFieldValue로 설정합니다.
기본값은 빈값으로 필요 시 DEXT5.config.UserFieldValue = "newValue"로 설정합니다.

sample code

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">  
        <script type="text/javascript">

            // 에디터 User 태그의 name을 dext5ID로 변경합니다.
            DEXT5.config.UserFieldID = "dext5ID";
            
            // 에디터 User 태그의 value를 dext5Val로 변경합니다.
            DEXT5.config.UserFieldValue = "dext5Val";

            new Dext5editor("editor1");

        </script>       
</div>
