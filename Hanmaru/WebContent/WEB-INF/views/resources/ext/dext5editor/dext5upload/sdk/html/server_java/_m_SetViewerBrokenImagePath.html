DEXT5 Upload :: SetViewerBrokenImagePath

void SetViewerBrokenImagePath(String viewerBrokenImagePath)

웹표준 모드의 뷰어사용시 뷰어에 사용되는 dext5upload/images/broken.png 파일의 경로를 설정합니다.
일반적으로 dext5upload/handler/dext5viewer.jsp를 다른 경로에 사용될 경우 설정합니다.
이 경우 dext5viewer.jsp 페이지에서 broken.png의 경로가 접근 가능하도록 설정해야 합니다.

return value

없음.

parameters

viewerBrokenImagePath  웹표준 모드의 뷰어사용시 뷰어에 사용되는 dext5upload/images/broken.png 파일의 경로를 의미합니다.

remarks

기본값 : ../images/broken.png ( dext5upload/images/broken.png )

sample code

Raonwiz.Dext5.UploadHandler upload = new Raonwiz.Dext5.UploadHandler();

upload.SetViewerBrokenImagePath("../images/broken.png");

String result = upload.Process(request, response, event);
