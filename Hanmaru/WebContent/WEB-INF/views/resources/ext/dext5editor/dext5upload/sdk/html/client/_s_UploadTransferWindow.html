DEXT5 Upload :: UploadTransferWindow

XML : <upload_transfer_window><view>.....</view></upload_transfer_window>
Config : config.UploadTransferWindow.View

Config 설정이 우선 적용됩니다.
업로드 전송창 UI를 설정합니다.

remarks

XML:view, Config:View  기본값은 "standard" 이고, 축소창은 "light" 입니다.
기본창 사이즈는 "470 X 260 (픽셀)"이고, 축소창 사이즈는 "470 X 170 (픽셀)" 입니다.

sample code

업로드 전송창을 축소창으로 설정합니다.

XML

.....
<setting>
    .....
    <upload_transfer_window>
        <view>light</view>
        .....
    </upload_transfer_window>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        DEXT5UPLOAD.config.UploadTransferWindow = {
            View: 'light'
        }

        new Dext5Upload("upload1");
    </script>
</div>
