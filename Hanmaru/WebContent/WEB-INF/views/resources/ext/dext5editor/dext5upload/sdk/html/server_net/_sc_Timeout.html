DEXT5 Upload :: Timeout

Timeout 설정

업로드 또는 다운로드 요청시 Timeout을 설정하게 되면 해당 시간 후 요청이 강제로 중지됩니다.
해당 설정에 따라 업로드할 때의 허용되는 시간이 결정됩니다.
업로드 또는 다운로드 요청이 해당 시간보다 오래걸릴 경우 서버에서 에러가 발생됩니다.

remarks

없음.

설정 방법

Web.config에서 설정

<?xml version="1.0" encoding="UTF-8" ?>
<configuration>
.....
    <system.web>
    .....
        <httpruntime executiontimeout="3600" maxrequestlength="2048000" /> <!-- 초단위로 설정 -->
    .....
    </system.web>
.....
</configuration>
