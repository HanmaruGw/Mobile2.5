DEXT5 Upload :: ShowButtonBarEdit

XML : <show_button_bar_edit>.....</show_button_bar_edit>, Config : config.ButtonBarEdit

Config 설정이 우선 적용됩니다.
업로드가 편집모드일 때 버튼바 영역에 노출 할 버튼을 설정합니다.

remarks

"0"으로 설정시 버튼 영역을 표시하지 않습니다.
(버튼영역 미표시는 IE 브라우저 모드 10 이상부터 가능합니다. 플러그인 모드는 IE브라우저 모드 상관 없이 사용 가능합니다.)

버튼 종류는
add : 파일추가
send : 전송하기
remove : 항목제거
remove_all : 전체 항목제거
move_first : 맨 앞으로
move_forward : 앞으로
move_back : 뒤로
move_end : 맨 뒤로

버튼 종류는 ",(콤마)"로 구분하여 설정합니다.

sample code

업로드가 편집모드일 때 버튼 바 영역에 파일추가,전송하기 버튼을 설정합니다.

XML

.....
<setting>
    .....
    <show_button_bar_edit>add,send</show_button_bar_edit>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        DEXT5UPLOAD.config.ButtonBarEdit = 'add,send';

        new Dext5Upload("upload1");
    </script>
</div>
