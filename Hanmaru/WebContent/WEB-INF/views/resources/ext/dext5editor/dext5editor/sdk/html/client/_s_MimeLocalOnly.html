DEXT5 Editor :: MimeLocalOnly

config.MimeLocalOnly

MIME 인코딩 대상 파일 중에서 외부(Local PC가 아닌)도 포함 할 지를 설정합니다.

remarks

기본값은 "1"로 PC에 위치한 파일만 인코딩, "0"으로 설정 시 로컬 PC 및 외부 URL에 있는 모든 파일 인코딩으로 설정합니다.

sample code

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        // 외부에 위치한 파일도 인코딩을 합니다.
        DEXT5.config.MimeLocalOnly = "0";

        new Dext5editor("editor1");

    </script>       
</div>
