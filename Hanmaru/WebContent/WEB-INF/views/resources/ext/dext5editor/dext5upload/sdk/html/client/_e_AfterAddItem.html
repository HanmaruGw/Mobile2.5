DEXT5 Upload :: AfterAddItem

void DEXT5UPLOAD_AfterAddItem(uploadID, strFileName, nFileSize, nAddItemIndex, objFile)

파일 추가 후 발생합니다.

return value

없음.

parameters

uploadID  파일 추가 완료된 업로드의 id를 의미합니다.
strFileName  추가될 파일명을 의미합니다.
nFileSize  파일 사이즈를 의미합니다.
nAddItemIndex  1부터 시작되는 파일의 순서정보를 의미합니다.
objFile  추가될 파일객체를 의미합니다.(html5모드)

remarks

DEXT5 Upload 설정 config.xml에서 use_after_add_event 값이 1일 경우 발생합니다.

            // 파일 추가 후 처리할 내용

        }
</script>
