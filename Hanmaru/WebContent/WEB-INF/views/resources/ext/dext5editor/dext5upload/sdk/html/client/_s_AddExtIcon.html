DEXT5 Upload :: AddExtIcon

XML : <add_ext_icon><icon>.....</icon>.....</add_ext_icon>

웹표준 모드인 경우 존재하지 않는 파일 확장자 icon의 경우 이미지 아이콘 파일 추가 후 설정할 수 있습니다.

remarks

icon 추가 설정 후 /dext5upload/images 폴더에 해당 icon 이미지를 "16 X 16 (픽셀)"의 크기의 파일로 적용해야 합니다.

sample code

ext,test 확장자에 대한 icon을 설정합니다.

XML

.....
<add_ext_icon>
    .....
    <icon>ext</icon>
    <icon>test</icon>
    .....
</add_ext_icon>
.....
