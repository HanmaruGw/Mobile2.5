DEXT5 Upload :: DropZoneAddItem

void DEXT5UPLOAD_DropZoneAddItem(objFile, strFilePath, objDropZoneId)

DropZone에 파일을 Drop 후 발생합니다.

return value

없음.

parameters

objFile  DropZone에 Drop된 파일객체를 의미합니다.(플러그인모드는 null)
strFilePath  DropZone에 Drop된 파일경로를 의미합니다.(웹표준모드는 null)
objDropZoneId  DropZone객체의 id를 의미합니다.

remarks

DEXT5 Upload 설정 config.xml에서 use_dropzone 값이 1일 경우 발생합니다.
DropZone으로 만들려는 객체의 class명에 "dext5dropzone"이 포함되어야 발생합니다.

sample code

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>	

<script type="text/javascript">
        function DEXT5UPLOAD_DropZoneAddItem(objFile, strFilePath, objDropZoneId) {
            // 업로드에 DropZone에서 Drop된 파일을 추가하는 API
            DEXT5UPLOAD.AddDropZoneFile(objFile, strFilePath, objDropZoneId);

        }
</script>
