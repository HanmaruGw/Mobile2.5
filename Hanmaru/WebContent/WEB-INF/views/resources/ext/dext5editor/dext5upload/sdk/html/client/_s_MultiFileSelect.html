DEXT5 Upload :: MultiFileSelect

XML : <multi_file_select>.....</multi_file_select>, Config : config.MultiFileSelect

Config 설정이 우선 적용됩니다.
업로드에서 파일 추가시 여러개 파일 선택 또는 한개씩만 선택 후 적용하도록 설정합니다.

remarks

기본값은 "1" 이고 여러 개의 파일 선택할 수 있도록 설정합니다. "0" 으로 설정시 한개씩만 설정합니다.
(IE 브라우저는 IE10 이상부터 여러개 파일 선택 설정이 가능합니다. 플러그인 모드는 IE브라우저 버전에 상관 없이 사용 가능합니다.)

sample code

파일선택 시 단일 파일 선택모드로 설정합니다.

XML

.....
<setting>
    .....
    <multi_file_select>0</multi_file_select>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        DEXT5UPLOAD.config.MultiFileSelect = '0';

        new Dext5Upload("upload1");
    </script>
</div>
