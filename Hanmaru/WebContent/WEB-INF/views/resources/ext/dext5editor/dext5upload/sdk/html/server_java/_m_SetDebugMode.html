DEXT5 Upload :: SetDebugMode

void SetDebugMode(boolean isDebugMode)

업로드,다운로드 등등의 모든 서버 동작 중 발생되는 로그를 콘솔 또는 서버로그에 출력하기 위하여 설정합니다.

return value

없음.

parameters

isDebugMode  로그 출력 여부를 의미합니다. true: 로그 출력, false: 로그 출력 안합

remarks

기본값 : false

sample code

Raonwiz.Dext5.UploadHandler upload = new Raonwiz.Dext5.UploadHandler();

upload.SetDebugMode(true);

String result = upload.Process(request, response, event);
