DEXT5 Upload :: SetFolderTransfer

void SetFolderTransfer(useFolderTransfer, uploadID);

폴더 구조로 업로드 할수 있도록 설정합니다.

return value

없음.

parameters

useFolderTransfer  0 : 폴더구조 업로드 사용안함
1 : 폴더구조 업로드 사용함(일반파일 업로드 사용)
2 : 폴더구조 업로드 사용함(일반파일 업로드 미사용)
uploadID  다운로드 경로를 설정할 업로드의 id를 의미합니다.

remarks

없음.

sample code

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>	
<script type="text/javascript">
        function setFolderTransfer(){
            // 폴더 구조 업로드 사용안함으로 설정합니다.
            DEXT5UPLOAD.SetFolderTransfer(0, 'upload1');
        }
</script>

        </script>
