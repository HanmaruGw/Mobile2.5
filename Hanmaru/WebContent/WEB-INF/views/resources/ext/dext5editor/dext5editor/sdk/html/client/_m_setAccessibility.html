DEXT5 Editor :: setAccessibility

void setAccessibility(value, editor_id)

에디터의 웹 접근성 단계를 설정합니다. (기본값은 0으로 설정되어 있습니다.)

return value

없음.

parameters

value  에디터에 설정할 웹 접근성 단계를 의미합니다.
editor_id  설정할 에디터의 id를 의미합니다.

remarks

에디터를 1개만 생성했을 경우 parameter를 입력하지 않아도 웹 접근성 단계를 설정할 수 있습니다.
예) id가 editor1인 에디터를 생성했을 경우
     DEXT5.setAccessibility('2', 'editor1'), DEXT5.setAccessibility('2') 모두 editor1 에디터의 웹 접근성을 2단계로 설정합니다.

에디터를 2개 이상 생성했을 경우 parameter를 반드시 입력해야 합니다.
예) id가 editor1, editor2인 에디터를 생성했을 경우
     DEXT5.setAccessibility('1', 'editor1')   editor1 에디터의 웹 접근성을 1단계로 설정합니다.
     DEXT5.setAccessibility('2', 'editor2')   editor2 에디터의 웹 접근성을 2단계로 설정합니다.
