DEXT5 Upload :: SetDebugMode

void SetDebugMode(bool isDebug, string debugFilePath)

업로드,다운로드 등등의 모든 서버 동작 중 발생되는 로그를 출력하기 위하여 사용합니다.
설정된 물리적 경로에 로그가 출력되어 파일로 생성됩니다.

return value

없음.

parameters

isDebug  로그 출력 여부를 의미합니다. true: 로그 출력, false: 로그 출력 안합
debugFilePath  로그가 출력 될 파일의 경로값을 의미합니다.

remarks

기본값 : false

sample code

Raonwiz.Dext5.UploadHandler upload = new Raonwiz.Dext5.UploadHandler();

upload.SetDebugMode(true, @"로그파일 경로");

upload.Process(context);
