DEXT5 Upload :: ResumeDownload

XML : <resume_mode download="....."></resume_mode>, Config : config.ResumeDownload

Config 설정이 우선 적용됩니다.
업로드 이어받기를 설정합니다.
파일 다운로드 중에 네트워크가 단절되거나 사용자 요청에 의해 중지 되어도 이어받기가 가능합니다.
웹표준모드, 플러그인모드 모두 지원합니다.

remarks

기본값은 "0" 이고, "1"로 설정시 이어받기가 가능합니다.

sample code

업로드 이어 받기를 설정합니다.

XML

.....
<setting>
    .....
    <resume_mode download="1"></resume_mode>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        DEXT5UPLOAD.config.ResumeDownload = '1';

        new Dext5Upload("upload1");
    </script>
</div>
