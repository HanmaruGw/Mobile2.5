DEXT5 Upload :: DoSaveAllFilesEx

string DoSaveAllFilesEx(zipFileName, uploadID)

모든 파일을 로컬에 저장할 때 zip파일로 저장합니다.

return value

string  빈값("")을 리턴합니다.

parameters

zipFileName  저장할 zip파일명을 의미합니다.
uploadID  실행할 업로드의 id를 의미합니다.

remarks

없음.
