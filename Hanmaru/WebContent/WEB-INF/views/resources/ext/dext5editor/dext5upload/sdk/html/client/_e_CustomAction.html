DEXT5 Upload :: CustomAction

void DEXT5UPLOAD_CustomAction(uploadID, strCmmand)

커스텀버튼 클릭 시 발생합니다.

return value

없음.

parameters

uploadID  커스텀버튼이 클릭된 업로드의 id를 의미합니다.
strCmmand  커스텀버튼의 command값을 의미합니다.(에디터에 커스텀버튼 추가 시 입력한 설정값)

remarks

없음.

sample code

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>	

<script type="text/javascript">
        function DEXT5UPLOAD_CustomAction(uploadID, strCmmand) {
            if (strCmmand == 'custom_button_action1') {    
            // 커스텀버튼1 클릭 시 처리할 내용

            } else if (strCmmand == 'custom_button_action2') {
            // 커스텀버튼2 클릭 시 처리할 내용

            }

        }
</script>

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">
