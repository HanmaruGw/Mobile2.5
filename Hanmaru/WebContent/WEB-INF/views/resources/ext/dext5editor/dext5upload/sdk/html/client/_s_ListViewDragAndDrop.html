DEXT5 Upload :: ListViewDragAndDrop

XML : <list_view_drag_and_drop>.....</list_view_drag_and_drop>, Config : config.ListViewDragAndDrop

Config 설정이 우선 적용됩니다.
업로드 영역에 드래그 앤 드랍해서 파일추가할 수 있도록 설정합니다.(편집 모드)

remarks

기본값은 "1" 이고 드래그 앤 드랍 파일 추가 적용, "0"으로 설정시 드래그 앤 드랍 파일추가 적용되지 않습니다.
(드래그 앤 드랍 파일추가 기능은 IE10 이상부터 가능합니다. 플러그인 모드는 IE버전에 상관 없이 사용 가능합니다.)

sample code

업로드 영역에 드래그 앤 드랍 파일추가 미 설정합니다.

XML

.....
<setting>
    .....
    <list_view_drag_and_drop>0</list_view_drag_and_drop>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        DEXT5UPLOAD.config.ListViewDragAndDrop = '0';

        new Dext5Upload("upload1");
    </script>
</div>
