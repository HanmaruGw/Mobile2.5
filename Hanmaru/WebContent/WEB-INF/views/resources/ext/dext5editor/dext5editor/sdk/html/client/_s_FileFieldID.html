DEXT5 Editor :: FileFieldID

config.FileFieldID

파일 업로드에 사용되는 File 태그의 name을 설정합니다.

remarks

기본값은 'Filedata' 로 설정됩니다.
필요시 DEXT5.config.FileFieldID = "newName" 으로 설정합니다.

sample code

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">  
        <script type="text/javascript">

            // 파일 업로드 File 태그의 name을 dext5로 변경합니다.
            DEXT5.config.FileFieldID = "dext5";

            new Dext5editor("editor1");

        </script>       
</div>
