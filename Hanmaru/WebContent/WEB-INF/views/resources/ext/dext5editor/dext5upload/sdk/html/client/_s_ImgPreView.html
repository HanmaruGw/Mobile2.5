DEXT5 Upload :: ImgPreView

XML : <img_preview>.....</img_preview>, Config : config.ImgPreView
XML : <img_preview width="....."></img_preview>, Config : config.ImgPreViewWidth
XML : <img_preview height"....."></img_preview>, Config : config.ImgPreViewHeight

Config 설정이 우선 적용됩니다.
이미지 미리보기 창 기능을 설정합니다.

remarks

기본값은 "0"이고 사용안함, "1"로 설정시 이미지 미리보기 창 사용입니다.

XML:width, Config:ImgPreViewWidth  이미지 미리보기 창의 넓이를 설정합니다.
기본값은 "320px" 입니다.
넓이는 "px"(픽셀)만 가능합니다.
XML:height, Config:ImgPreViewHeight  이미지 미리보기 창의 높이를 설정합니다.
기본값은 "280px" 입니다.
높이는 "px"(픽셀)만 가능합니다.

sample code

이미지 미리보기 창 사용으로 설정합니다.
이미지 미리보기 창 넓이를 250px 로 설정합니다.
이미지 미리보기 창 넓이를 200px 로 설정합니다.

XML

.....
<setting>
    .....
    <img_preview width="250px" height="200px">1</img_preview>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        DEXT5UPLOAD.config.ImgPreView = '1';
        DEXT5UPLOAD.config.ImgPreViewWidth = '250px';
        DEXT5UPLOAD.config.ImgPreViewHeight = '200px';

        new Dext5Upload("upload1");
    </script>
</div>
