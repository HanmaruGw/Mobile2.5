DEXT5 Upload :: BeforeDeleteItem

boolean DEXT5UPLOAD_BeforeDeleteItem(uploadID, strWebFile, strItemKey, strItemUrlOrPath, nDeleteItemIndex)

파일 삭제 전 발생합니다.

return value

boolean  true일 경우 파일을 삭제하고, false일 경우 삭제하지 않습니다.

parameters

uploadID  파일 삭제 요청한 업로드의 id를 의미합니다.
strWebFile  Web파일과 Local 파일의 구분 값을 의미합니다. (0:local, 1:web)
strItemKey  파일의 Unique Key를 의미합니다.
strItemUrlOrPath  파일의 저장위치를 의미합니다.
nDeleteItemIndex  1부터 시작되는 파일의 순서정보를 의미합니다.

remarks

DEXT5 Upload 설정 config.xml에서 use_delete_event의 값이 1일 경우 발생합니다.

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>	

<script type="text/javascript">
    function DEXT5UPLOAD_BeforeDeleteItem(uploadID, strWebFile, strItemKey, strItemUrlOrPath, nDeleteItemIndex) {
        // 파일 삭제 전 처리할 내용

           return true or false;
    }
</script>
