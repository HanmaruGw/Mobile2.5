DEXT5 Upload :: FolderTransfer

XML : <folder_transfer>.....</folder_transfer>, Config : config.FolderTransfer

Config 설정이 우선 적용됩니다.
폴더 구조로 업로드를 설정합니다.
폴더 상태로 폴더를 추가, 폴더 안에 있는 파일들을 한번에 추가 합니다.

remarks

기본값은 "0"이고, "1"로 설정시 폴더 구조로 업로드 설정합니다.

sample code

폴더 구조로 업로드를 설정합니다.

XML

.....
<setting>
    .....
    <folder_transfer>1</folder_transfer>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>
	
<!-- ..... 생략 ..... -->
