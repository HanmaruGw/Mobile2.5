DEXT5 Upload :: GetTotalFileCount

int GetTotalFileCount(uploadID)

업로드 영역에 추가된 전체 파일개수를 가져옵니다.

return value

int  추가된 전체 파일개수를 의미합니다.

parameters

uploadID  추가된 전체 파일개수를 가져올 업로드의 id를 의미합니다.

remarks

없음.
