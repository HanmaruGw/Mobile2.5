DEXT5 Upload :: ChunkSize

XML : <chunk_size unit=".....">.....</chunk_size>, Config : config.ChunkSize

Config 설정이 우선 적용됩니다.
HTML5가 지원되는 브라우저에서만 설정합니다.
파일을 전송할때, 분할 전송 크기를 설정합니다.

remarks

XML:chunk_size  기본값은 "1MB"(1048576 BYTE)이고, "사용자 설정값" 으로 설정합니다.
파일 단위는 chunk_size unit 속성값에 B(byte), KB(kilobyte), MB(megabyte), GB(gigabyte) 으로 설정합니다.
Config:ChunkSize  기본값은 "1MB"(1048576 BYTE)이고, "사용자 설정값" 으로 설정합니다.
파일 단위는 B(byte), KB(kilobyte), MB(megabyte), GB(gigabyte) 으로 설정합니다.

sample code

업로드할 때 해당 파일 크기를 5MB로 설정합니다.

XML

.....
<uploader_setting>
    .....
    <chunk_size unit="MB">5</chunk_size>
    .....
</uploader_setting>
.....

Config

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        DEXT5UPLOAD.config.ChunkSize = '5MB';

        new Dext5Upload("upload1");
    </script>
</div>
