DEXT5 Editor :: StatusBarLoading

XML : <show_statusbar loading="....."></show_statusbar>, Config : config.StatusBarLoading

Config 설정이 우선 적용됩니다.
에디터의 보기화면(statusBar)에서 "디자인", "HTML" 및 "미리보기", "TEXT" 탭 변경 시 로딩 텍스트 노출 여부를 설정합니다.

remarks

"0"으로 설정 시 로딩 텍스트를 미노출 하고, "1"로 설정 시 로딩 텍스트를 노출 합니다. 기본값은 "1" 입니다.

sample code

탭 변경 시 로딩 텍스트 미노출로 설정합니다.

XML

.....
<setting>
    .....
    <show_statusbar loading="0"></show_statusbar>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">  
    <script type="text/javascript">

        DEXT5.config.StatusBarLoading = "0";

        new Dext5editor("editor1");

    </script>       
</div>
