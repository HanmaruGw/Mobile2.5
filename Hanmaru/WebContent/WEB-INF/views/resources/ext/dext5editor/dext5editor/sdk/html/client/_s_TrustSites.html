DEXT5 Editor :: TrustSites

XML : <trust_sites>.....</trust_sites>, Config : config.TrustSites

Config 설정이 우선 적용됩니다.
신뢰할 수 있는 사이트를 추가합니다.

sample code

"www.dext5.com"을 신뢰할 수 있는 사이트에 추가합니다.

XML

.....
<license>
    .....
    <trust_sites>www.dext5.com</trust_sites>
    .....
</license>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.TrustSites = "www.dext5.com";

        new Dext5editor("editor1");

    </script>       
</div>
