DEXT5 Upload :: SilentUpload

XML : <upload_transfer_window><silent_upload>.....</silent_upload></upload_transfer_window>
Config : config.SilentUpload

Config 설정이 우선 적용됩니다.
전송창 노출 여부를 설정합니다.

remarks

기본값은 "0" 노출이고, "1"로 설정시 미노출입니다.

sample code

업로드 전송창을 미노출합니다.

XML

.....
<setting>
    .....
    <upload_transfer_window>
        .....
        <silent_upload>1</silent_upload>
        .....
    </upload_transfer_window>
    .....
</setting>
.....

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>
