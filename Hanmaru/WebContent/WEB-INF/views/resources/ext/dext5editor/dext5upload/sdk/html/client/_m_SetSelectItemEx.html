DEXT5 Upload :: SetSelectItemEx

void SetSelectItemEx(uniqueKey, appendMode, uploadID)

업로드 영역 외부에서 파일을 선택합니다.

return value

없음.

parameters

uniqueKey  선택하려는 파일의 uniqueKey 값을 의미합니다.
(DEXT5 Upload API인 AddUploadedFile의 첫번째 파라미터 값)을 의미합니다.
appendMode  0 – 기존 선택된 파일 선택해제
1 – 기존 선택된 파일 유지
uploadID  적용할 업로드의 id값을 의미합니다.

uniqueKey가 빈 값일 경우
appendMode  0 – 전체 파일 선택해제
1 – 전체 파일 선택

remarks

없음.
