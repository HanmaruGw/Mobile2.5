DEXT5 Upload :: DoSelectFolder

void DoSelectFolder(uploadID)

폴더 선택 다이얼로그 화면을 보여줍니다.

return value

없음.

parameters

uploadID  실행할 업로드의 id를 의미합니다.

remarks

업로드 모드가 upload일 때 동작 합니다.(업로드모드)

        function doSelectFolder() {
            DEXT5UPLOAD.DoSelectFolder("upload1");
        }
</script>
