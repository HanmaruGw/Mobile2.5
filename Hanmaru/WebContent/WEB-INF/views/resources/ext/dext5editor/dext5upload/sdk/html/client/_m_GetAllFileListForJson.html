DEXT5 Upload :: GetAllFileListForJson

object GetAllFileListForJson(uploadID)

전송완료 후 새롭게 업로드 된 파일과 이미 업로드된 파일 리스트 모두를 json 형태로 리턴합니다.

return value

object  전송완료 후 새롭게 업로드 된 파일과 이미 업로드된 파일 리스트 모두를 의미합니다.

parameters

uploadID  파일 리스트를 가져올 업로드의 id를 의미합니다.

remarks

없음.
