DEXT5 Editor :: TableClassList

XML : <class_list>.....</class_list>
Config : config.TableClassList

Config 설정이 우선 적용됩니다.
테이블에 지정 가능한 class명을 설정합니다.

remarks

콤마(,)로 구분하여 설정합니다.

sample code

테이블에 지정 가능한 class명을 "tableClass"로 설정합니다.

XML

.....
<setting>
    .....
    <table_property>
        .....
        <class_list>tableClass</class_list>
        .....
    </table_property>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>
