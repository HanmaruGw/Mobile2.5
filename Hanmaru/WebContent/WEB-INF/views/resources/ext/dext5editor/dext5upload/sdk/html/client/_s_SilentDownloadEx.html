DEXT5 Upload :: SilentDownload

XML : <upload_transfer_window><silent_download_ex>.....</silent_download_ex></upload_transfer_window>
Config : config.SilentDownloadEx

Config 설정이 우선 적용됩니다.
웹표준 모드시 다운로드 진행 팝업창 노출과 서버에서 처리되는 다운로드 된 파일사이즈 체크 여부를 설정합니다.
설정시 after_download_event, finish_downloaded_event, download_multi 기능 사용이 불가능합니다.

remarks

기본값은 "0" 노출 및 체크이고, "1"로 설정시 미노출 및 체크안함 입니다.

sample code

웹표준 모드시 다운로드 전송창을 미노출하고 서버에서 처리되는 다운로드 된 파일사이즈 체크하지 않습니다.

XML

.....
<setting>
    .....
    <upload_transfer_window>
        .....
        <silent_download_ex>1</silent_download_ex>
        .....
    </upload_transfer_window>
    .....
</setting>
.....
