DEXT5 Upload :: AutomaticConnection

XML : <automatic_connection>.....</automatic_connection>, Config : config.AutomaticConnection

Config 설정이 우선 적용됩니다.
업로드/다운로드 시 묻지 않고 계속 업로드/다운로드 시도 사용을 설정합니다.
(불안정한 네트워크 환경 발생 시)

remarks

기본값은 "0"이고, "1"로 설정시 묻지 않고 계속 업로드/다운로드를 시도합니다.
config.ResumeUpload / config.ResumeDownload 설정값이 모두 "1" 일 경우 적용됩니다.

sample code

업로드/다운로드 시 묻지 않고 계속 시도 사용을 설정합니다.

XML

.....
<setting>
    .....
    <automatic_connection>1</automatic_connection>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">

</div>
