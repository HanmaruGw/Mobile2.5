DEXT5 Editor :: PasteImageBase64

XML : <paste_image_base64>.....</paste_image_base64>

이미지를 붙여넣기 할 때 DEXT5 Handler를 사용하지 않고, IMG의 src 값을 base64로 처리합니다.
이 기능은 html5 spec을 지원하는 브라우저에서만 사용이 가능합니다.

remarks

기본값은 "0" 이며 사용하지 않음 (DEXT5 Handler를 사용하여 서버에 업로드한 이미지 경로를 사용),
"1" 로 설정 시 base64 사용으로 설정합니다.

sample code

이미지를 붙여넣기 할 때 IMG의 src 값을 base64로 처리되도록 설정합니다.

.....
<extra_setting>
    <paste_image_base64>1</paste_image_base64>
    .....
</extra_setting>
.....
