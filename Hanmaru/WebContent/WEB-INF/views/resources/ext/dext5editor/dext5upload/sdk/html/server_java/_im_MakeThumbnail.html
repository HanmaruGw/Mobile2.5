DEXT5 Upload :: MakeThumbnail

String MakeThumbnail
(String strSourceFile, String strSuffix, int nNewWidth, int nNewHeight, boolean bDeleteOldFile)

파일 업로드 시 이미지 파일인 경우 동일 폴더에 이미지에 대한 Thumbnail 처리를 할 경우 사용됩니다.
이벤트 UploadCompleteBeforeEvent, UploadCompleteEvent 에서 사용이 가능합니다.

return value

String  Thumbnail 처리된 이미지의 물리적 경로를 의미합니다.

parameters

strSourceFile  이미지 파일의 물리적 경로를 의미합니다.
strSuffix  Thumbnail 처리된 이미지의 파일명 뒤에 붙을 문자열을 의미합니다.
nNewWidth  Thumbnail의 Width 값을 의미합니다.
nNewHeight  Thumbnail의 Height 값을 의미합니다.
bDeleteOldFile  원본 파일의 삭제 여부를 설정합니다. true: 삭제, false: 삭제안함

remarks

없음.
