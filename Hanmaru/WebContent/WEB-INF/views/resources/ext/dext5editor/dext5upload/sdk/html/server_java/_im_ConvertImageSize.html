DEXT5 Upload :: ConvertImageSize

void ConvertImageSize(String strSourceFile, int nNewWidth, int nNewHeight)

파일 업로드 시 이미지 파일인 경우 이미지 크기를 변환할 경우 사용됩니다.
이벤트 UploadCompleteBeforeEvent, UploadCompleteEvent 에서 사용이 가능합니다.

return value

없음.

parameters

strSourceFile  이미지 파일의 물리적 경로를 의미합니다.
nNewWidth  이미지의 크기를 변환할 Width를 의미합니다.
nNewHeight  이미지의 크기를 변환할 Height를 의미합니다.

remarks

없음.
