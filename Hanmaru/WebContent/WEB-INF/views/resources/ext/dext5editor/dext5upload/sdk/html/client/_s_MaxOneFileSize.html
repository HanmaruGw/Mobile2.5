DEXT5 Upload :: MaxOneFileSize

XML : <max_one_file_size unit=".....">.....</max_one_file_size>, Config : config.MaxOneFileSize

Config 설정이 우선 적용됩니다.
업로드할 때 파일당 용량제한을 설정합니다.

remarks

XML:max_one_file_size  기본값은 "0" 용량 제한없음이고, "사용자 설정값" 으로 설정합니다.
파일 단위는 max_one_file_size의 unit 속성값에 B(byte), KB(kilobyte), MB(megabyte), GB(gigabyte) 으로 설정합니다.
Config:MaxOneFileSize  기본값은 "0" 용량 제한없음이고, "사용자 설정값" 으로 설정합니다.
파일 단위는 B(byte), KB(kilobyte), MB(megabyte), GB(gigabyte) 으로 설정합니다.

sample code

업로드 할 파일당 최대 크기를 10MB로 제한합니다.

XML

.....
<setting>
    .....
    <max_one_file_size unit="MB">10</max_one_file_size>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>
