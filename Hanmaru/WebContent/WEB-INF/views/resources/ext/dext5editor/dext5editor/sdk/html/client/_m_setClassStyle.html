DEXT5 Editor :: setClassStyle

void setClassStyle(editor_id, className)

에디터 선택영역에 class명을 적용합니다.

return value

없음.

parameters

editor_id  접근할 에디터의 id 입니다.
className  적용할 class명 입니다.

remarks

없음.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
<script type="text/javascript">

    function setClassStyle(){
        // 에디터에 작성 된 모든 내용을 선택합니다.
        DEXT5.selectAll('editor1');

        // 선택된 내용에 class명을 적용시킵니다.
        DEXT5.setClassStyle('editor1', 'dext_class_name');
    }

</script>	

<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px;">
  
    <!-- 에디터 생성 -->  
    <script type="text/javascript">
        new Dext5editor('editor1');
    </script>

</div>
