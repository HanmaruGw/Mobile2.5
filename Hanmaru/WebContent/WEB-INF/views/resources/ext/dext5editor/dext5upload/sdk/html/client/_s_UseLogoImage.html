DEXT5 Upload :: UseLogoImage

XML : <upload_transfer_window><use_logo_image>.....</use_logo_image></upload_transfer_window>
XML : .....<use_logo_image logo_path="....."></use_logo_image>.....
XML : .....<use_logo_image logo_ver="....."></use_logo_image>.....

Config 설정이 우선 적용됩니다.
업로드 전송창 Logo를 설정합니다.

remarks

use_logo_image  기본값은 "1" 이고, Logo 숨김 설정은 "0" 입니다.
logo_path  Logo Image 경로를 설정합니다.
logo_ver  Logo Image 버전을 관리합니다.

sample code

업로드 전송창 Logo를 사용으로 설정하고, Logo의 경로를 "http://www.dext5.com/image/logo.png"로 설정하고, Logo의 버전을 2로 설정합니다.

XML

.....
<setting>
    .....
    <upload_transfer_window>
        .....
        <use_logo_image logo_path="http://www.dext5.com/image/logo.png" logo_ver="2">1</use_logo_image>
        .....
    </upload_transfer_window>
    .....
</setting>
.....
