DEXT5 Upload :: BorderStyle

XML : <border_style>.....</border_style>, Config : config.BorderStyle

Config 설정이 우선 적용됩니다.
업로드 영역에 추가된 파일들을 구분할 선 스타일을 설정합니다.

remarks

기본값은 "none" 입니다.
선 스타일 종류는 "실선 : solid, 점선 : dashed, 선없음 : none" 입니다.

sample code

업로드 영역의 파일 구분 선을 실선으로 설정합니다.

XML

.....
<setting>
    .....
    <border_style>solid</border_style>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        DEXT5UPLOAD.config.BorderStyle = 'solid';

        new Dext5Upload("upload1");
    </script>
</div>
