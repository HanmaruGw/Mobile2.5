DEXT5 Upload :: UploadMethodHtml4

XML : <upload_method_html4>.....</upload_method_html4>
Config : config.UploadMethodHtml4

XML : <upload_method_html4 html4_limit_file_size="....."></upload_method_html4>
Config : config.Html4LimitFileSize

Config 설정이 우선 적용됩니다.
XML:upload_method_html4, Config:UploadMethodHtml4  HTML4환경에서 업로드모드를 설정합니다.
XML:html4_limit_file_size, Config:Html4LimitFileSize  html4 모드인 경우 서버에서 개별 파일 Max Size Check 여부를 설정합니다.

remarks

XML:upload_method_html4, Config:UploadMethodHtml4  기본값은 "1" html4모드이고, "0"으로 설정시 swf(플래시) 모드를 사용합니다.
XML:html4_limit_file_size, Config:Html4LimitFileSize  기본값은 "0" 서버에서 개별 파일 Max Size Check를 하지 않고,
"1"로 설정시 서버에서 개별 파일 Max Size Check함

sample code

HTML4환경에서 업로드를 html4 모드로 설정하고, 서버에서 개별 파일 Max Size Check 여부를 check로 설정 합니다.

XML

.....
<uploader_setting>
    .....
    <upload_method_html4 html4_limit_file_size="1">1</upload_method_html4>
    .....
</uploader_setting>
.....
