DEXT5 Editor :: ShowToolBar

XML : <show_toolbar>.....</show_toolbar>, Config : config.ShowToolBar

Config 설정이 우선 적용됩니다.
에디터의 툴바는 상단과 하단으로 구성되어 있습니다.
이 설정은 상단, 하단 툴바를 보이거나 숨기는 기능을 설정합니다.
사용 용도에 따라서 상단, 하단 툴바를 숨김으로 설정하기를 원할 때 사용하면 유용 할 수 있습니다.

remarks

기본값은 "3" 이고 "상단, 하단" 툴바 모두 보임으로 설정합니다.
"2" 로 설정 시 "상단" 툴바를 숨김으로 설정합니다.
"1" 로 설정 시 "하단" 툴바를 숨김으로 설정합니다.
"0" 으로 설정 시 "상단, 하단" 툴바 모두를 숨김으로 설정합니다.

sample code

에디터 툴바 중 상단 툴바를 숨김으로 설정합니다.

XML

.....
<setting>
    .....
    <show_toolbar>2</show_toolbar>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">  
    <script type="text/javascript">

        DEXT5.config.ShowToolBar = "2";

        new Dext5editor("editor1");
    </script>       

</div>
