DEXT5 Upload :: SetUploadMode

void SetUploadMode(mode, uploadID)

DEXT5UPLOAD의 모드를 설정합니다.

return value

없음.

parameters

mode  DEXT5UPLOAD의 모드를 의미합니다.(edit or 빈값:편집모드, view:보기모드)
uploadID  설정할 업로드의 id를 의미합니다.

remarks

없음.
