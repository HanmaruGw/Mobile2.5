DEXT5 Upload :: UseDeleteEndTimeEvent

XML : <use_delete_end_time_event>.....</use_delete_end_time_event>
Config : config.UseDeleteEndTimeEvent

Config 설정이 우선 적용됩니다.
업로드 리스트 컨트롤에서 파일 삭제 후 이벤트 함수 사용을 설정합니다.

remarks

기본값은 "1" 사용이고, "0"으로 설정시 이벤트 미사용합니다.
"1"으로 설정 시 파일 삭제 후에 DEXT5UPLOAD_DeleteItemEndTime(uploadID, nDeleteItemCount) 함수가 호출 됩니다.

sample code

파일 삭제 후 이벤트 함수 미사용으로 설정합니다.

XML

.....
<setting>
    .....
    <use_delete_end_time_event>0</use_delete_end_time_event>
    .....
</setting>
.....
