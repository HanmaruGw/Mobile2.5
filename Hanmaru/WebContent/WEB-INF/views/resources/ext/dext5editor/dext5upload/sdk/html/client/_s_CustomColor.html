DEXT5 Upload :: CustomHeaderColor, CustomFooterColor
DEXT5 Upload :: CustomProgressBarColor, CustomTextColor

XML : <skin_name color1="....."></skin_name>, Config : config.CustomHeaderColor
XML : <skin_name color2="....."></skin_name>, Config : config.CustomFooterColor
XML : <skin_name color3="....."></skin_name>, Config : config.CustomProgressBarColor
XML : <skin_name text_color="....."></skin_name>, Config : config.CustomTextColor

Config 설정이 우선 적용됩니다.
업로드 색상을 사용자가 원하는 색상으로 변경합니다.

remarks

CustomHeaderColor  업로드창 (상단/ 하단 / 하단 버튼 바탕색)을 설정합니다.
CustomFooterColor   업로드창 (상단 / 하단 버튼 텍스트색)을 설정합니다.
CustomProgressBarColor  전송창(상단/ 전체 전송률/ 전송 용량률/ 취소 버튼 바탕색)을 설정합니다.
CustomTextColor  전송창 (상단 / 하단 취소 버튼 텍스트 색)을 설정합니다.

#을 포함한 hex값으로 입력합니다.

sample code

사용자가 원하는 색상으로 설정합니다.

XML

.....
<setting>
    .....
    <skin_name color1="#ff0000" color2="#f7f140" color3="#33e8f5" text_color="#c2ffd0">.....</skin_name>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        DEXT5UPLOAD.config.CustomHeaderColor = '#ff0000';
        DEXT5UPLOAD.config.CustomFooterColor = '#f7f140';
        DEXT5UPLOAD.config.CustomProgressBarColor = '#33e8f5';
        DEXT5UPLOAD.config.CustomTextColor = '#c2ffd0';

        new Dext5Upload("upload1");
    </script>
</div>
