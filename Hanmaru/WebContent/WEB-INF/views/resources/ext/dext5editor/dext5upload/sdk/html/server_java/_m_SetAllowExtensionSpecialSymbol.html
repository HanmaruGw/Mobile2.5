DEXT5 Upload :: SetAllowExtensionSpecialSymbol

void SetAllowExtensionSpecialSymbol(String allowExtensionSpecialSymbol)

서버에서 업로드 또는 다운로드시 파일에 대한 확장자 체크를 할 경우
해당 확장자 값 중 특수문자를 모두 빈값으로 치환합니다.(보안상의 이슈로 인하여 치환)
특정 특수문자에 대하여 빈값으로 치환되지 않도록 하기 원할 경우 설정합니다.(여러개일 경우 ,로 구분하여 설정)

return value

없음.

parameters

allowExtensionSpecialSymbol  확장자 값 중 빈값으로 치환되지 않도록 하기 원하는 특정 특수문자를 의미합니다.

remarks

기본값 : 빈값

sample code

Raonwiz.Dext5.UploadHandler upload = new Raonwiz.Dext5.UploadHandler();

upload.SetAllowExtensionSpecialSymbol("_,*");

String result = upload.Process(request, response, event);
