DEXT5 Editor :: RemoveContextItem

XML : <remove_context_item>.....</remove_context_item>, Config : config.RemoveContextItem

Config 설정이 우선 적용됩니다.
에디터 우클릭 메뉴에서 미표시 하려는 command를 설정합니다.

remarks

XML:remove_context_item  필요 시 <item>.....</item> 항목을 추가하여 설정합니다.
Config:RemoveItem  필요 시 DEXT5.config.RemoveContextItem = "command명";에 콤마(,)로 구분하여 설정합니다.

sample code

우클릭 메뉴 중 붙여넣기를 미표시로 설정합니다.

XML

.....
<remove_context_item>
    <item>paste</item>
</remove_context_item>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>
