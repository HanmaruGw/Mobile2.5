DEXT5 Upload :: DownloadHandlerUrl

XML : <download_handler_url>.....</download_handler_url>, Config : config.DownloadHandlerUrl

Config 설정이 우선 적용됩니다.
사용 용도에 따라서 다운로드 경로를 상황에 맞게 개별 Download Handler를 구현하여 사용하고자 할 때 설정합니다.

remarks

설정하실 때 http:// 부터의 전체경로를 설정합니다.

sample code

업로드 Download handler url을 http://www.dext5.com/handler/dext5handler.jsp로 설정합니다.

XML

.....
<uploader_setting>
    .....
    <download_handler_url>http://www.dext5.com/handler/dext5handler.jsp</download_handler_url>
    .....
</uploader_setting>
.....

Config

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        DEXT5UPLOAD.config.DownloadHandlerUrl = 'http://www.dext5.com/handler/dext5handler.jsp';

        new Dext5Upload("upload1");
    </script>
</div>
