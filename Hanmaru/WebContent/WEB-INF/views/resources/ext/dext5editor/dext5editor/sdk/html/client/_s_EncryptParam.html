DEXT5 Editor :: EncryptParam

XML : <security><encrypt_param>.....</encrypt_param></security>
Config : config.Security.EncryptParam

Config 설정이 우선 적용됩니다.
에디터에서 파일을 업로드 할 경우 파라미터 암호화에 대한 설정입니다.

remarks

기본값은 "0" 이고 암호화 하지 않음, "1" 로 설정 시 암호화합니다.

sample code

에디터에서 파일을 업로드 할 경우 파라미터 암호화 설정합니다.

XML

.....
<setting>
    .....
    <security>
        <encrypt_param>1</encrypt_param>
    </security>
    .....
</setting>
.....
