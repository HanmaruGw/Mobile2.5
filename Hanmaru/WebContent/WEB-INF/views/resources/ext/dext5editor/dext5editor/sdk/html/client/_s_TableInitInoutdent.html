DEXT5 Editor :: TableInitInoutdent

XML : <table_property init_inoutdent="....."></table_property>
Config : config.TableInitInoutdent

Config 설정이 우선 적용됩니다.
표 생성 시 들여쓰기 할 기본 px값을 설정합니다.

remarks

기본값은 "20"px 이고, "사용자 설정값"으로 설정합니다.

sample code

표 생성 시 들여쓰기 할 기본 px값을 30px로 설정합니다.

XML

.....
<setting>
    .....
    <table_property init_inoutdent="30">
        .....
    </table_property>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">  
    <script type="text/javascript">

        DEXT5.config.TableInitInoutdent = "30";

        new Dext5editor("editor1");

    </script>       
</div>
