DEXT5 Upload :: ListViewDbclick

XML : <list_view_dbclick>.....</list_view_dbclick>, Config : config.ListViewDbclick

Config 설정이 우선 적용됩니다.
업로드 영역을 더블클릭 이벤트에 의해 파일선택 창이 열리도록 설정합니다.(편집 모드)

remarks

기본값은 "1"이고 더블클릭 이벤트를 적용, "0" 으로 설정시 더블클릭 이벤트가 적용되지 않습니다.
(IE 브라우저 모드는 IE10이상부터 가능합니다. 플러그인 모드는 IE브라우저 모드 상관 없이 사용 가능합니다.)

sample code

업로드 영역에 더블클릭 이벤트를 미 설정합니다.

XML

.....
<setting>
    .....
    <list_view_dbclick>0</list_view_dbclick>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        DEXT5UPLOAD.config.ListViewDbclick = '0';

        new Dext5Upload("upload1");
    </script>
</div>
