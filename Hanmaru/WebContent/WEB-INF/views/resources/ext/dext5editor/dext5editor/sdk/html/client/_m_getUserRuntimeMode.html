DEXT5 Editor :: getUserRuntimeMode

object getUserRuntimeMode(editor_id)

에디터의 Runtime Mode를 가져옵니다.

return value

에디터의 Runtime Mode를 리턴합니다.

parameters

editor_id  접근할 에디터의 id를 의미합니다.

remarks

없음.
