DEXT5 Editor :: SetValueObjId

config.SetValueObjId

textarea에 ID 값을 설정하면 에디터 로드 후 textarea 내용이 에디터에 세팅됩니다.

remarks

필요 시 DEXT5.config.SetValueObjId = "ID 값" 으로 설정합니다.

sample code

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">   
    <script type="text/javascript">

        // 에디터 로드 후 ID가 "dext5" 인 textarea의 내용을 에디터에 세팅합니다.
        DEXT5.config.SetValueObjId = "dext5";

        new Dext5editor("editor1");

    </script>       
</div>
