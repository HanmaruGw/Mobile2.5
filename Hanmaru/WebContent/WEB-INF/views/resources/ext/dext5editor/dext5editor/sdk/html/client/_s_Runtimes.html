DEXT5 Editor :: Runtimes

XML : <runtimes>.....</runtimes>, Config : config.Runtimes

Config 설정이 우선 적용됩니다.
에디터 모드를 설정합니다.
(웹표준모드, 플러그인모드 중 선택)

remarks

html5 설정 시 html5 모드가 우선 적용됩니다.
ieplugin 설정 시 ieplugin 모드가 우선 적용되며, 지원되지 않는 브라우저는 html5 모드가 적용됩니다.(ie 전용 모드)
versionieplugin 설정 시 적용한 버전 이하에서만 플러그인 모드로 적용됩니다.(ie 전용 모드)
예) 9ieplugin 으로 적용하면 ie9 이하 버전에서 플러그인 모드로 적용됩니다.
예) "html5" or "ieplugin" or versionieplugin

sample code

에디터를 ieplugin 모드로 설정합니다.

XML

.....
<setting>
    <runtimes>ieplugin</runtimes>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">   
    <script type="text/javascript">

        DEXT5.config.Runtimes = "ieplugin";

        new Dext5editor("editor1");

    </script>       
</div>
