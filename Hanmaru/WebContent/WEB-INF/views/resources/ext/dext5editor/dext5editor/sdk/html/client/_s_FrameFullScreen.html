DEXT5 Editor :: FrameFullScreen

XML : <frame_fullscreen>.....</frame_fullscreen>, Config : config.FrameFullScreen

Config 설정이 우선 적용됩니다.
프레임일 경우 풀 스크린 아이콘 버튼 사용여부를 설정합니다.

remarks

기본값은 "0" 이고, "1" 로 설정 시 프레임일 경우 풀 스크린 아이콘 버튼을 보여줍니다.

sample code

풀 스크린 아이콘 버튼 사용으로 설정합니다.

XML

.....
<setting>
    .....
    <frame_fullscreen>1</frame_fullscreen>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.FrameFullScreen = "1";

        new Dext5editor("editor1");

    </script>       
</div>
