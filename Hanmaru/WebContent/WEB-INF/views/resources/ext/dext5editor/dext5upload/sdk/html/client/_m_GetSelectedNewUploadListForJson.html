DEXT5 Upload :: GetSelectedNewUploadListForJson

object GetSelectedNewUploadListForJson(uploadID)

선택된 파일중 사용자가 추가한 파일 리스트를 json 형태로 리턴합니다.

return value

object  선택된 파일중 사용자가 추가한 파일 리스트를 의미합니다.

parameters

uploadID  파일 리스트를 가져올 업로드의 id를 의미합니다.

remarks
