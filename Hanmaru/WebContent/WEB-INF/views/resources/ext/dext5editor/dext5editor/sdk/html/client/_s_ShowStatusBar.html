DEXT5 Editor :: ShowStatusBar

XML : <show_statusbar>.....</show_statusbar>, Config : config.ShowStatusBar

Config 설정이 우선 적용됩니다.
에디터의 보기화면(statusBar)은 "디자인", "HTML", "미리보기", "TEXT" 탭으로 구성되어 있습니다.
이 설정은 보기화면(statusBar)을 보이거나 숨기는 기능을 설정합니다.
사용 용도에 따라서 상태바 숨김으로 설정을 원할 때 사용하면 유용 할 수 있습니다.

remarks

기본값은 "1" 이고 상태바 보임, "0" 로 설정시 숨김으로 설정합니다.

sample code

에디터 보기화면(statusBar)을 숨김으로 설정합니다.

XML

.....
<setting>
    .....
    <show_statusbar>0</show_statusbar>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">  
    <script type="text/javascript">

        DEXT5.config.ShowStatusBar = "0";

        new Dext5editor("editor1");

    </script>       
</div>
