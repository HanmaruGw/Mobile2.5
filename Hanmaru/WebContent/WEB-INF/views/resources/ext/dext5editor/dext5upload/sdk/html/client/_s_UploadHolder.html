DEXT5 Upload :: UploadHolder

config.UploadHolder

업로드를 특정 영역에 생성할 수 있도록 합니다.

remarks

업로드를 생성할 영역의 id값을 입력합니다.
필요시 DEXT5UPLOAD.config.UploadHolder = "아이디값" 으로 설정합니다.

sample code

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        // 업로드 생성할 객체의 ID를 설정합니다.
        DEXT5UPLOAD.config.UploadHolder = "upload_area";

        new Dext5Upload("upload1");
    </script>
</div>
