DEXT5 Editor :: DefaultMessage

config.DefaultMessage

에디터 디자인 영역에 기본으로 띄울 메세지를 설정합니다.

remarks

디자인 영역을 클릭하거나, 디자인 영역에서 키보드 입력을 하면 Message가 사라집니다.

sample code

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">    
    <script type="text/javascript">
            
        // 에디터에 메세지를 띄워줍니다.
        DEXT5.config.DefaultMessage = "<span>Default Message!! 클릭하면 사라집니다.</span>";

        new Dext5editor("editor1");
            
    </script>       
</div>
