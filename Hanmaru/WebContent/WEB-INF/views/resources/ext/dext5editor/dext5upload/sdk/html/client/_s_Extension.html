DEXT5 Upload :: Extension

XML : <extension allow_or_limit="....."></extension>, Config : config.ExtensionAllowOrLimit
XML : <extension>.....</extension>, Config : config.ExtensionArr

Config 설정이 우선 적용됩니다.
업로드 할 파일 확장자를 허용 또는 제한 설정(XML:allow_or_limit, Config:ExtensionAllowOrLimit) 후
제한 또는 허용할 파일 확장자를 설정(XML:extension, Config:ExtensionArr)합니다.

remarks

XML:allow_or_limit, Config:ExtensionAllowOrLimit  설정값이 "0" 이면 설정한 파일 확장자명만 "제한"으로 설정되며, "1"로 설정시 "허용"으로 설정됩니다.
XML:extension, Config:ExtensionArr  제한 및 허용할 확장자명을 ",(콤마)"로 구분하여 설정합니다.

sample code

업로드 할 파일 확장자를 제한으로 설정합니다.
제한할 파일 확장자를 txt,gif로 설정합니다.

XML

.....
<setting>
    .....
    <extension allow_or_limit="0">txt,gif</extension>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        DEXT5UPLOAD.config.ExtensionAllowOrLimit = '0';
        DEXT5UPLOAD.config.ExtensionArr = 'txt,gif';

        new Dext5Upload("upload1");
    </script>
</div>
