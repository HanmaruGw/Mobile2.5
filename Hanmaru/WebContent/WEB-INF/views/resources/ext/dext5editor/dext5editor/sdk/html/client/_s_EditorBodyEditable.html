DEXT5 Editor :: EditorBodyEditable

XML : <editor_body_editable>.....</editor_body_editable>, Config : config.EditorBodyEditable

Config 설정이 우선 적용됩니다.
디자인 영역에 작성 가능 여부를 설정합니다.

remarks

기본값은 "1" 이고 작성 가능, "0"으로 설정 시 작성 불가능으로 설정합니다.

sample code
