DEXT5 Editor :: dext_paste_image_event

void dext_paste_image_event(imgArr)

에디터에 이미지 붙여넣기 시 발생합니다.

return value

없음.

parameters

imgArr  붙여넣기 한 이미지객체 배열입니다.

remarks

에디터의 설정파일(dext5editor/config/dext_editor.xml)의 event_for_paste_image 설정값이 1인 경우에 이벤트가 발생합니다.

    }
