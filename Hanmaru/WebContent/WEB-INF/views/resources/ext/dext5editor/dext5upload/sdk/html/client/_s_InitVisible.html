DEXT5 Upload :: InitVisible

config.InitVisible

업로드를 화면에 표시할지 여부를 설정합니다.

remarks

"true"로 설정 시 업로드를 화면에 표시하고, "false"로 설정 시 표시하지 않습니다.
기본값은 "true" 입니다.

sample code

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:400px">  
    <script type="text/javascript">
        // upload1 업로드는 화면에 표시가 됩니다.
        DEXT5UPLOAD.config.InitVisible = true;

        new Dext5Upload("upload1");
    </script>

    <script type="text/javascript">
        // upload2 업로드는 화면에 표시되지 않습니다.
        DEXT5UPLOAD.config.InitVisible = false;

        new Dext5Upload("upload2");
    </script>
</div>
