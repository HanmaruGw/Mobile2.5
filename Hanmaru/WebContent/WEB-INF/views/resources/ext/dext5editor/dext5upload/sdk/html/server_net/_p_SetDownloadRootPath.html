DEXT5 Upload :: SetDownloadRootPath

upload.SetDownloadRootPath

클라이언트에서 다운로드 또는 열기 경로 설정시 경로의 앞부분 일부를 서버에서 설정할 경우 사용합니다.
다운로드 또는 열기의 모든 경로에 고정적으로 설정되므로 모든 파일의 경로가 동일할 경우 사용해야 합니다.

remarks

기본값 : 빈값

sample code

Client

<script type="text/javascript">
    function DEXT5UPLOAD_OnCreationComplete(uploadID) {
        // 실제 경로는 /download/전경사진.jpg 이라고 가정
        DEXT5UPLOAD.AddUploadedFile('1', '전경사진.jpg', '/전경사진.jpg', '87325', '전경사진.jpg', 'dext5upload');
    }
</script>

Server

Raonwiz.Dext5.UploadHandler upload = new Raonwiz.Dext5.UploadHandler();
upload.SetDownloadRootPath = "/download";
upload.Process(context);
