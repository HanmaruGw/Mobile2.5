DEXT5 Upload :: UseSizeColumn

XML : <use_size_column>.....</use_size_column>, Config : config.SizeColumnWidth

Config 설정이 우선 적용됩니다.
업로드 영역 항목 중 파일크기 항목 노출 여부를 설정합니다.

remarks

설정값 "0" 파일크기 항목 노출안함이고, "1"로 설정시 파일크기 항목을 노출합니다.

sample code

파일크기 항목 노출안함으로 설정합니다.

XML

.....
<setting>
    .....
    <use_size_column>0</use_size_column>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        DEXT5UPLOAD.config.SizeColumnWidth = '0';

    </script>
</div>
