DEXT5 Upload :: MaxRequestLength

MaxRequestLength 설정

요청되는 ContentLength의 최대값을 설정할 때 사용됩니다.
해당 설정에 따라 업로드할 때의 허용되는 파일사이즈가 결정됩니다.
해당 설정보다 큰 ContentLength값이 업로드될 경우 서버에서 에러가 발생됩니다.

remarks

없음.

설정 방법

Web.config에서 설정

IIS6

<?xml version="1.0" encoding="UTF-8" ?>
<configuration>
.....
    <system.web>
    .....
        <httpruntime executiontimeout="3600" maxrequestlength="2048000" /> <!-- MByte 단위로 설정 -->
    .....
    </system.web>
.....
</configuration>

IIS7

<?xml version="1.0" encoding="UTF-8" ?>
<configuration>
.....
    <system.web>
    .....
        <httpruntime executiontimeout="3600" maxrequestlength="2048000" /> <!-- MByte 단위로 설정 -->
    .....
    </system.web>
    <system.webserver>
    .....
        <security>
            .....
            <requestfiltering>
                <requestlimits maxallowedcontentlength="2048000000" /> <!-- Byte 단위로 설정 -->
            </requestfiltering>
        </security>
    .....
    </system.webserver>
.....
</configuration>
