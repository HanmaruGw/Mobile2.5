DEXT5 Editor :: NextTabElementId

config.NextTabElementId

Tab키를 입력하면 마지막 포커스가 설정한 아이디로 이동 합니다.
(단, 포커스를 줄 객체 아이디를 설정 시)

remarks

필요시 DEXT5.config.NextTabElementId = "아이디값" 으로 설정합니다.

sample code

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">  
    <script type="text/javascript">

        // 마지막 포커스가 설정한 ID값 dext5 으로 이동합니다.
        DEXT5.config.NextTabElementId = "dext5";

        new Dext5editor("editor1");

    </script>       
</div>
