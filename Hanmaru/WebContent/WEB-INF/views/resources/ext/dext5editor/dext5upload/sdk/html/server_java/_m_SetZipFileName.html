DEXT5 Upload :: SetZipFileName

void SetZipFileName(String zipFileName)

웹표준 모드에서 멀티 파일 다운로드시 zip 파일명을 설정합니다.

return value

없음.

parameters

zipFileName  웹표준 모드에서 멀티 파일 다운로드시 zip 파일의 파일명을 의미합니다.

remarks

기본값 : download.zip

sample code

Raonwiz.Dext5.UploadHandler upload = new Raonwiz.Dext5.UploadHandler();

upload.SetZipFileName("download.zip");
