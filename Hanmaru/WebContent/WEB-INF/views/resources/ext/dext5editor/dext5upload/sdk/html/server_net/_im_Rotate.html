DEXT5 Upload :: Rotate

void Rotate(string pSourceFileFullPath, int pDegree)

파일 업로드 시 이미지 파일인 경우 이미지를 회전할 경우 사용됩니다.
이벤트 UploadCompleteBeforeEvent, UploadCompleteEvent 에서 사용이 가능합니다.

return value

없음.

parameters

pSourceFileFullPath  이미지 파일의 물리적 경로를 의미합니다.
pDegree  이미지를 회전할 각도를 의미합니다.

remarks

없음.

string newFileLocation = parameterEntity.NewFileLocation;
