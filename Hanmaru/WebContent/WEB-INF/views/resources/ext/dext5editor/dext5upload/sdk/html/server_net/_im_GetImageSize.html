DEXT5 Upload :: GetImageSize

Size GetImageSize(string pSourceFileFullPath)

파일 업로드 시 이미지 파일인 경우 이미지의 크기정보를 얻으려고 할 때 사용됩니다.
이벤트 UploadCompleteBeforeEvent, UploadCompleteEvent 에서 사용이 가능합니다.

return value

System.Drawing.Size  이미지의 크기정보를 의미합니다.

parameters

pSourceFileFullPath  이미지 파일의 물리적 경로를 의미합니다.

remarks

없음.

sample code

string newFileLocation = parameterEntity.NewFileLocation;
using (Raonwiz.Dext5.Common.Dext5Image dextImage = new Raonwiz.Dext5.Common.Dext5Image())
{
    try
    {
        string sourceFileFullPath = newFileLocation;
        
        System.Drawing.Size size = dextImage.GetImageSize(sourceFileFullPath);
        int _width = size.Width;
        int _height = size.Height;

    }
    catch (Exception ex) { }
}
