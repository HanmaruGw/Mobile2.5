DEXT5 Upload :: UserMessage

XML : <user_message edit="....."></user_message>, Config : config.UserMessage.Edit
XML : <user_message view="....."></user_message>, Config : config.UserMessage.View

Config 설정이 우선 적용됩니다.
업로드 편집 / 보기 모드의 버튼 영역 왼쪽 또는 오른쪽에 문구 입력을 설정합니다.

remarks

기본값은 "0" 사용안함이고, "1"로 설정시 사용입니다.

Edit  업로드 편집 모드 버튼 영역에 문구를 설정합니다.
View 업로드 보기 모드 버튼 영역에 문구를 설정합니다.

버튼 위치가 왼쪽이면 문구 삽입은 오른쪽이 되며, 버튼 위치가 오른쪽이면 문구 삽입은 왼쪽에 위치합니다.
"사용자 설정값"으로 문구를 입력합니다.

sample code

편집/보기 모드의 텍스트 문구를 설정합니다.

XML

.....
<setting>
    .....
    <user_message edit="편집 모드 노출 문구" view="보기 모드 노출 문구"></user_message>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        DEXT5UPLOAD.config.UserMessage = {
            Edit: '편집 모드 노출 문구',
            View: '보기 모드 노출 문구'
        }

        new Dext5Upload("upload1");
    </script>
</div>
