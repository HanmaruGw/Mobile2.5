[플러그인 전용]

DEXT5 Upload :: UseScriptEventControl

XML : <use_script_event_control>.....</use_script_event_control>, Config : config.UseScriptEventControl

Config 설정이 우선 적용됩니다.
dext5upload의 이벤트에서 플러그인 모드에서 발생하는 이벤트의 수신 여부를 설정합니다.

remarks

기본값은 "1" 플러그인 모드에서 발생하는 이벤트를 javacript로 보내 dext5upload의 이벤트에서 받을 수 있고, "0"으로 설정시 미사용입니다.

sample code

dext5upload의 이벤트에서 플러그인 모드에서 발생하는 이벤트를 받지 않도록 설정합니다.

XML

.....
<setting>
    .....
    <use_script_event_control>0</use_script_event_control>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5upload/js/dext5upload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        DEXT5UPLOAD.config.UseScriptEventControl = '0';

        new Dext5Upload("upload1");
    </script>
</div>
