DEXT5 Editor :: dext_editor_loaded_event

void dext_editor_loaded_event(editor)

에디터가 로드 완료(작업준비) 되었을 때 발생합니다.

return value

없음.

parameters

editor   이벤트를 발생시킨 editor 객체입니다.

remarks

에디터가 작업 준비가 완료되었을 때 발생하므로 여러가지 초기값을 설정하려면 이곳에서 설정을 합니다.
