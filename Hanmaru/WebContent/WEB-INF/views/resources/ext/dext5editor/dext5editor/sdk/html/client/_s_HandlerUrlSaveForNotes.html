DEXT5 Editor :: HandlerUrlSaveForNotes

XML : <handler_url_save_for_notes>.....</handler_url_save_for_notes>
Config : config.HandlerUrlSaveForNotes

Config 설정이 우선 적용됩니다.
IBM NOTES 서버를 사용할 때 DEXT5 Editor의 서버 로직을 사용할 수 없기 때문에 IBM NOTES서버에서 파일을 받을 경로를 설정합니다.

remarks

http://를 포함한 전체 경로로 설정합니다.

sample code

IBM NOTES서버에서 파일을 받을 경로를 설정합니다.

XML

.....
<uploader_setting>
    .....
    <handler_url_save_for_notes>http://www.dext5.com/notes_handler.jsp</handler_url_save_for_notes>
    .....
</uploader_setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">  
    <script type="text/javascript">

        DEXT5.config.HandlerUrlSaveForNotes = "http://www.dext5.com/notes_handler.jsp";

    </script>       
</div>
