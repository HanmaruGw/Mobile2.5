한글 입력 문제(Firefox 41.0버전)

현상

Firefox 41.0 버전에서 DEXT5 Editor에 한글 입력 시 줄 바꿈 후 텍스트를 입력하기 시작하면 커서가 이전 줄로 이동, 일정 시간 후 커서가 텍스트 맨 첫 줄로 강제 이동 등이 모든 HTML 에서 발생 하고 있습니다.

원인

Firefox 에서 한글 입력 문제는 41.0버전에 새롭게 추가된 윈도비스타 이후 OS의 TSF를 사용한 향상된 IME 지원 기능인 Text Services Framework이 원인으로 밝혀져 있습니다.

해결 방법

Firefox 42.0 버전으로 업그레이드를 하시면 됩니다. (자동 업그레이드 지원 안함)
