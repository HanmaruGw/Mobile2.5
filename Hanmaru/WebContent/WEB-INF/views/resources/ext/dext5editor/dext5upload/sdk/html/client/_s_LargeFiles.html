DEXT5 Upload :: LargeFiles

XML : <large_files unit=".....">.....</large_files>, Config : config.LargeFiles.Size
XML : <large_files max_count="....."></large_files>, Config : config.LargeFiles.MaxCount
XML : <large_files max_total_size="....."></large_files>, Config : config.LargeFiles.MaxTotalSize
XML : <large_files text="....."></large_files>, Config : config.LargeFiles.Text
XML : <large_files color="....."></large_files>, Config : config.LargeFiles.Color

Config 설정이 우선 적용됩니다.
업로드 기준 파일 크기 이상이면 대용량 텍스트 노출 기능을 설정합니다.

remarks

XML:large_files, Config:LargeFiles.Size  기본값 "0" 이면 대용량 텍스트 미노출이고, 사용자 설정값"으로 설정시 대용량 텍스트를 노출합니다.
XML:파일 단위는 large_files unit 속성값에 B(byte), KB(kilobyte), MB(megabyte), GB(gigabyte) 으로 설정합니다.
Config:파일 단위는 B(byte), KB(kilobyte), MB(megabyte), GB(gigabyte) 으로 설정합니다.
XML:max_count, Config:LargeFiles.MaxCount  최대 대용량 파일 개수 제한 값
XML:max_total_size, Config:LargeFiles.MaxTotalSize  최대 전체 대용량 파일 용량 값
XML:파일 단위는 large_files unit 속성값에
B(byte), KB(kilobyte), MB(megabyte), GB(gigabyte) 으로 설정합니다.
Config:파일 단위는 B(byte), KB(kilobyte), MB(megabyte), GB(gigabyte) 으로 설정합니다.
XML:text, Config:LargeFiles.Text  대용량 문구 변경 값
XML:color, Config:LargeFiles.Color  대용량 텍스트 컬러 값

setting 항목 중 (최대 파일 개수 제한 값(max_total_file_count), 업로드 된 파일의 용량 제한 값(max_total_file_size))이
대용량 설정 속성 값(최대 파일 개수 제한 값(max_count),총 업로드 된 파일의 용량 제한 값(max_total_size))보다
최우선으로 적용 됩니다.

예) 최대 파일 개수 제한 값(max_total_file_count)이 5 이고, 최대 대용량 파일 개수 제한 값(max_count)이 10 이라면
최대 파일 개수 제한 값(max_total_file_count)인 5가 우선 적용 됩니다.

sample code

업로드 할 파일 크기가 10MB 이상이면 '대용량2' 라는 텍스트가 #ff0000 색상으로 노출되어 생성합니다.
대용량 파일에 대한 Max Count는 10개로, Max Total Size는 500MB로 설정합니다.

XML

.....
<setting>
    .....
    <large_files unit="MB" max_count="10" max_total_size="500" text="대용량2" color="#ff0000">10</large_files>
    .....
</setting>
.....
